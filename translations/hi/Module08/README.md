# Module 08: Microsoft Foundry Local के साथ व्यावहारिक अनुभव - संपूर्ण डेवलपर टूलकिट

## परिचय

Microsoft Foundry Local एज AI विकास की अगली पीढ़ी का प्रतिनिधित्व करता है, जो डेवलपर्स को स्थानीय स्तर पर AI एप्लिकेशन बनाने, तैनात करने और स्केल करने के लिए शक्तिशाली उपकरण प्रदान करता है, साथ ही Azure AI Foundry के साथ सहज एकीकरण बनाए रखता है। यह मॉड्यूल Foundry Local की स्थापना से लेकर उन्नत एजेंट विकास तक का व्यापक कवरेज प्रदान करता है।

**मुख्य तकनीकें:**
- Microsoft Foundry Local CLI और SDK
- Azure AI Foundry एकीकरण
- ऑन-डिवाइस मॉडल इन्फरेंस
- स्थानीय मॉडल कैशिंग और अनुकूलन
- एजेंट-आधारित आर्किटेक्चर

## मॉड्यूल के सीखने के उद्देश्य

इस मॉड्यूल को पूरा करने के बाद, आप:

- **Foundry Local सेटअप में महारत हासिल करेंगे**: Windows 11 विकास के लिए Foundry Local को स्थापित, कॉन्फ़िगर और अनुकूलित करें
- **विविध मॉडल तैनात करें**: CLI कमांड का उपयोग करके phi, qwen, deepseek, और GPT-OSS-20B मॉडल को स्थानीय रूप से चलाएं
- **उत्पादन समाधान बनाएं**: उन्नत प्रॉम्प्ट इंजीनियरिंग और डेटा एकीकरण के साथ AI एप्लिकेशन बनाएं
- **ओपन-सोर्स इकोसिस्टम का लाभ उठाएं**: Hugging Face मॉडल और समुदाय-चालित योगदान को एकीकृत करें
- **AI आर्किटेक्चर की तुलना करें**: LLMs और SLMs के बीच व्यापार-ऑफ और तैनाती रणनीतियों को समझें
- **AI एजेंट विकसित करें**: Foundry Local की आर्किटेक्चर और ग्राउंडिंग क्षमताओं का उपयोग करके बुद्धिमान एजेंट बनाएं
- **मॉडल को टूल्स के रूप में लागू करें**: एंटरप्राइज़ एप्लिकेशन के लिए मॉड्यूलर, अनुकूलन योग्य AI समाधान बनाएं

## पूर्व आवश्यकताएँ

### सिस्टम आवश्यकताएँ
- **ऑपरेटिंग सिस्टम**: Windows 11 (22H2 या बाद का संस्करण)
- **मेमोरी**: 16GB RAM (बड़े मॉडल के लिए 32GB अनुशंसित)
- **स्टोरेज**: मॉडल कैशिंग के लिए 50GB खाली स्थान
- **हार्डवेयर**: NPU-सक्षम डिवाइस अनुशंसित (Copilot+ PC), GPU वैकल्पिक
- **नेटवर्क**: प्रारंभिक मॉडल डाउनलोड के लिए हाई-स्पीड इंटरनेट

### विकास पर्यावरण
- Visual Studio Code AI Toolkit एक्सटेंशन के साथ
- Python 3.10+ और pip
- संस्करण नियंत्रण के लिए Git
- PowerShell या Command Prompt
- Azure CLI (क्लाउड एकीकरण के लिए वैकल्पिक)

### ज्ञान आवश्यकताएँ
- AI/ML अवधारणाओं की बुनियादी समझ
- कमांड लाइन से परिचित
- Python प्रोग्रामिंग की मूल बातें
- REST API अवधारणाएँ
- प्रॉम्प्टिंग और मॉडल इन्फरेंस का बुनियादी ज्ञान

## सीखने के परिणाम

इस मॉड्यूल को पूरा करने के बाद, आप सक्षम होंगे:

### तकनीकी महारत
- **तैनात और प्रबंधित करें**: विकास और उत्पादन वातावरण में Foundry Local स्थापना
- **मॉडल एकीकृत करें**: Microsoft, Hugging Face, और सामुदायिक स्रोतों से विविध मॉडल परिवारों के साथ सहजता से काम करें
- **एप्लिकेशन बनाएं**: उन्नत सुविधाओं और अनुकूलन के साथ उत्पादन-तैयार AI एप्लिकेशन बनाएं
- **एजेंट विकसित करें**: ग्राउंडिंग, तर्क, और टूल एकीकरण के साथ परिष्कृत AI एजेंट लागू करें

### रणनीतिक समझ
- **आर्किटेक्चर निर्णय**: स्थानीय बनाम क्लाउड तैनाती के बीच सूचित विकल्प बनाएं
- **प्रदर्शन अनुकूलन**: विभिन्न हार्डवेयर कॉन्फ़िगरेशन में इन्फरेंस प्रदर्शन को अनुकूलित करें
- **एंटरप्राइज़ स्केलिंग**: स्थानीय प्रोटोटाइप से एंटरप्राइज़ तैनाती तक स्केल करने वाले एप्लिकेशन डिज़ाइन करें
- **गोपनीयता और सुरक्षा**: स्थानीय इन्फरेंस के साथ गोपनीयता-संरक्षण AI समाधान लागू करें

### नवाचार क्षमताएँ
- **त्वरित प्रोटोटाइपिंग**: AI एप्लिकेशन अवधारणाओं को जल्दी से बनाएं और परीक्षण करें
- **सामुदायिक एकीकरण**: ओपन-सोर्स मॉडल का लाभ उठाएं और इकोसिस्टम में योगदान दें
- **उन्नत पैटर्न**: RAG, एजेंट, और टूल एकीकरण सहित अत्याधुनिक AI पैटर्न लागू करें
- **भविष्य-तैयार विकास**: उभरती AI तकनीकों और पैटर्न के लिए तैयार एप्लिकेशन बनाएं

## शुरुआत करें

1. **अपना पर्यावरण तैयार करें**: अनुशंसित हार्डवेयर विनिर्देशों के साथ Windows 11 सुनिश्चित करें
2. **पूर्व आवश्यकताएँ स्थापित करें**: विकास उपकरण और निर्भरताएँ सेट करें
3. **सत्र 1 से शुरू करें**: Foundry Local स्थापना और बुनियादी सेटअप से शुरुआत करें
4. **क्रमिक रूप से प्रगति करें**: इष्टतम सीखने की प्रगति के लिए सत्रों को क्रम में पूरा करें
5. **लगातार अभ्यास करें**: व्यावहारिक अभ्यास और परियोजनाओं के माध्यम से अवधारणाओं को लागू करें

## नमूनों के लिए त्वरित शुरुआत

### 1) पर्यावरण सेटअप (Windows cmd.exe)
```cmd
cd Module08
py -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

### 2) स्थानीय मॉडल शुरू करें (नया टर्मिनल)
```cmd
foundry model list
foundry model run phi-4-mini
```

### 3) Chainlit डेमो चलाएँ (सत्र 4)
```cmd
cd Module08
.\.venv\Scripts\activate
chainlit run samples\04\app.py -w
```

### 4) मल्टी-एजेंट कोऑर्डिनेटर चलाएँ (सत्र 5)
```cmd
cd Module08
.\.venv\Scripts\activate
python -m samples.05.agents.coordinator
```

यदि आपको कनेक्शन त्रुटियाँ दिखाई दें, तो Foundry Local को मान्य करें:
```cmd
curl http://localhost:8000/v1/models
```

### राउटर कॉन्फ़िगरेशन (सत्र 6)
राउटर एक त्वरित स्वास्थ्य जांच करता है और env-आधारित कॉन्फ़िग का समर्थन करता है:
```cmd
set BASE_URL=http://localhost:8000
set GENERAL_MODEL=phi-4-mini
set REASONING_MODEL=deepseek-r1-distill-qwen-7b
set CODE_MODEL=qwen2.5-7b-instruct
REM Or provide a full JSON registry
set TOOL_REGISTRY={"general":{"model":"phi-4-mini"}}
python samples\06\router.py "Pick the best model for code refactoring"
```
