## နိဒါန်း

EdgeAI အတွက် စတင်လေ့လာသူများအတွက် လမ်းညွှန်စာတမ်းမှ ကြိုဆိုပါသည်။ ဒီစာတမ်းက သင်ကြားမှုအထောက်အကူဖြစ်စေဖို့၊ သင်ကြားမှုအတွေ့အကြုံကို အများဆုံးရရှိစေဖို့ အကောင်းဆုံးနည်းလမ်းများကို ပံ့ပိုးပေးထားပါတယ်။ 

ဒီသင်တန်းက EdgeAI နည်းပညာများအပေါ် အခြေခံသိမှတ်စရာများကို အချိန်တိုအတွင်း သင်ကြားပေးမယ့် 20 နာရီသာ ကြာမြင့်တဲ့ သင်တန်းတစ်ခုဖြစ်ပြီး အလုပ်ရှုပ်နေတဲ့ ပရော်ဖက်ရှင်နယ်များနဲ့ ကျောင်းသားများအတွက် အထူးသင့်လျော်ပါတယ်။

## သင်တန်းအကျဉ်းချုပ်

ဒီသင်တန်းကို အောက်ပါ ခုနစ်ခုသော Module များအဖြစ် စီစဉ်ထားပါတယ်-

1. **EdgeAI အခြေခံအကြောင်းအရာများနှင့် ပြောင်းလဲမှု** - နည်းပညာအခြေခံနှင့် ပြောင်းလဲမှုကို နားလည်ခြင်း
2. **Small Language Model အခြေခံအဆောက်အအုံများ** - SLM မျိုးစိတ်များနှင့် အဆောက်အအုံများကို လေ့လာခြင်း
3. **Small Language Model Deployment** - Deployment နည်းလမ်းများကို လက်တွေ့ကျကျ အသုံးချခြင်း
4. **Model Format Conversion နှင့် Quantization** - OpenVINO အပါအဝင် Framework 6 ခုဖြင့် အဆင့်မြှင့်ခြင်း
5. **SLMOps - Small Language Model Operations** - ထုတ်လုပ်မှုဆိုင်ရာ စီမံခန့်ခွဲမှု
6. **SLM Agentic Systems** - AI Agent များနှင့် Function Calling
7. **EdgeAI Implementation Samples** - AI Toolkit နှင့် Platform-specific Implementation
