## အကျဉ်းချုပ်

EdgeAI အတွက် အခြေခံသင်ခန်းစာများကို ကြိုဆိုပါတယ်! ဒီစာရွက်စာတမ်းက သင့်ကို သင်ခန်းစာများကို ထိရောက်စွာ လေ့လာနိုင်ဖို့အတွက် လမ်းညွှန်ပေးပြီး သင့်ရဲ့ သင်ယူမှု အတွေ့အကြုံကို အများဆုံးရရှိစေဖို့ ရည်ရွယ်ပါတယ်။ ဒီစာရွက်စာတမ်းမှာ သင်ကြားမှုလမ်းကြောင်းများ၊ အကြံပြုထားသော သင်ကြားမှု အချိန်ဇယားများ၊ အဓိကအကြောင်းအရာ အကျဉ်းချုပ်များနှင့် EdgeAI နည်းပညာများကို နက်နက်ရှိုင်းရှိုင်း နားလည်စေရန် အထောက်အကူပြုသော အပိုဆောင်းအရင်းအမြစ်များ ပါဝင်ပါတယ်။

ဒီသင်ခန်းစာက ၂၀ နာရီသာ ကြာမြင့်ပြီး EdgeAI အကြောင်း အရေးကြီးသော အသိပညာများကို အချိန်တိုအတွင်း သင်ယူနိုင်အောင် ဖန်တီးထားတာဖြစ်ပြီး အလုပ်ရှုပ်နေတဲ့ ပရော်ဖက်ရှင်နယ်များနှင့် ကျောင်းသားများအတွက် အထူးသင့်လျော်ပါတယ်။

## သင်ခန်းစာ အကျဉ်းချုပ်

ဒီသင်ခန်းစာကို Module ခုနစ်ခုအဖြစ် စီစဉ်ထားပါတယ်-

1. **EdgeAI အခြေခံနှင့် ပြောင်းလဲမှု** - အဓိကအကြောင်းအရာများနှင့် နည်းပညာ ပြောင်းလဲမှုကို နားလည်ခြင်း
2. **Small Language Model အခြေခံများ** - SLM မျိုးစဉ်များနှင့် ၎င်းတို့၏ ဖွဲ့စည်းပုံများကို လေ့လာခြင်း
3. **Small Language Model တပ်ဆင်မှု** - လက်တွေ့ တပ်ဆင်မှု မဟာဗျူဟာများကို အကောင်အထည်ဖော်ခြင်း
4. **မော်ဒယ် ဖော်မတ် ပြောင်းလဲခြင်းနှင့် Quantization** - OpenVINO အပါအဝင် Framework ၆ ခုဖြင့် အဆင့်မြှင့်ခြင်း
5. **SLMOps - Small Language Model Operations** - ထုတ်လုပ်မှု လုပ်ငန်းစဉ် စီမံခန့်ခွဲမှုနှင့် တပ်ဆင်မှု
6. **SLM Agentic Systems** - AI Agents, Function Calling နှင့် Model Context Protocol
7. **EdgeAI Implementation Samples** - AI Toolkit, Windows Development နှင့် Platform-specific Implementation
8. **Microsoft Foundry Local – Developer Toolkit အပြည့်အစုံ** - Local-first Development နှင့် Azure Integration (Module 08)

### အဓိက သင်ယူရမည့် ရည်မှန်းချက်များ

- Cloud-based AI နှင့် Edge-based AI အကြား ကွာခြားချက်များကို နားလည်ပါ
- အရင်းအမြစ် အကန့်အသတ်ရှိသော ပတ်ဝန်းကျင်များအတွက် အဓိက အဆင့်မြှင့်နည်းလမ်းများကို ကျွမ်းကျင်ပါ
- EdgeAI နည်းပညာများ၏ လက်တွေ့အသုံးချမှုများကို ခွဲခြားပါ
- EdgeAI Project များအတွက် Development Environment တည်ဆောက်ပါ

### သင်ယူရမည့် အဓိကအချက်များ

#### အပိုင်း ၁: EdgeAI အခြေခံ
- **အဓိက အကြောင်းအရာများ**: 
  - Edge vs. Cloud Computing Paradigms
  - Model Quantization နည်းလမ်းများ
  - Hardware Acceleration ရွေးချယ်မှုများ (NPUs, GPUs, CPUs)
  - Privacy နှင့် Security အကျိုးကျေးဇူးများ

- **အပိုဆောင်းစာရွက်စာတမ်းများ**:
  - [TensorFlow Lite Documentation](https://www.tensorflow.org/lite)
  - [ONNX Runtime GitHub](https://github.com/microsoft/onnxruntime)
  - [Edge Impulse Documentation](https://docs.edgeimpulse.com)

#### အပိုင်း ၂: လက်တွေ့ ကိစ္စလေ့လာမှုများ
- **အဓိက အကြောင်းအရာများ**: 
  - Microsoft Phi & Mu Model Ecosystem
  - စက်မှုလုပ်ငန်းများအတွင်း လက်တွေ့အသုံးချမှုများ
  - တပ်ဆင်မှုအတွက် စဉ်းစားရမည့်အချက်များ

#### အပိုင်း ၃: လက်တွေ့ အကောင်အထည်ဖော်မှု လမ်းညွှန်
- **အဓိက အကြောင်းအရာများ**: 
  - Development Environment တည်ဆောက်ခြင်း
  - Quantization နှင့် Optimization Tools
  - EdgeAI Implementation များအတွက် အကဲဖြတ်နည်းလမ်းများ

#### အပိုင်း ၄: Edge Deployment Hardware
- **အဓိက အကြောင်းအရာများ**: 
  - Hardware Platform များကို နှိုင်းယှဉ်ခြင်း
  - Hardware-specific Optimization Strategies
  - Deployment အတွက် စဉ်းစားရမည့်အချက်များ

### ကိုယ်တိုင် စစ်ဆေးမေးခွန်းများ

1. Cloud-based AI နှင့် Edge-based AI တပ်ဆင်မှုများကို နှိုင်းယှဉ်ပါ။
2. Edge Deployment အတွက် မော်ဒယ်များကို အဆင့်မြှင့်ရန် အဓိကနည်းလမ်း ၃ ခုကို ရှင်းပြပါ။
3. AI မော်ဒယ်များကို Edge မှာ အလုပ်လုပ်စေခြင်း၏ အဓိက အကျိုးကျေးဇူးများက ဘာလဲ။
4. မော်ဒယ်ကို Quantize လုပ်ခြင်း၏ လုပ်ငန်းစဉ်နှင့် ၎င်း၏ Performance အပေါ် သက်ရောက်မှုကို ရှင်းပြပါ။
5. EdgeAI Deployment အတွက် Hardware Accelerators (NPUs, GPUs, CPUs) များက ဘယ်လို သက်ရောက်မှုရှိသလဲ။
