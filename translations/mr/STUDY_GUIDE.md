## परिचय

EdgeAI नवशिक्यांसाठी अभ्यास मार्गदर्शकात आपले स्वागत आहे! हा दस्तऐवज आपल्याला अभ्यास साहित्य प्रभावीपणे नेव्हिगेट करण्यास आणि शिकण्याचा अनुभव जास्तीत जास्त करण्यास मदत करण्यासाठी तयार केला आहे. यात संरचित शिकण्याचे मार्ग, सुचवलेले अभ्यास वेळापत्रक, मुख्य संकल्पना सारांश आणि Edge AI तंत्रज्ञानाची सखोल समज मिळवण्यासाठी पूरक संसाधने समाविष्ट आहेत.

हा 20 तासांचा संक्षिप्त कोर्स आहे जो EdgeAI बद्दल आवश्यक ज्ञान वेळेच्या कार्यक्षम स्वरूपात प्रदान करतो, जो व्यस्त व्यावसायिक आणि विद्यार्थ्यांसाठी योग्य आहे जे या उदयोन्मुख क्षेत्रात व्यावहारिक कौशल्ये पटकन मिळवू इच्छितात.

## कोर्सचा आढावा

हा कोर्स आठ व्यापक मॉड्यूल्समध्ये आयोजित केला आहे:

0. **EdgeAI परिचय** - उद्योग अनुप्रयोग आणि शिकण्याच्या उद्दिष्टांसह पाया आणि संदर्भ सेट करणे  
1. **EdgeAI मूलभूत गोष्टी आणि परिवर्तन** - मुख्य संकल्पना आणि तंत्रज्ञानातील बदल समजून घेणे  
2. **लहान भाषा मॉडेल पायाभूत तत्त्वे** - विविध SLM कुटुंबे आणि त्यांची आर्किटेक्चर एक्सप्लोर करणे  
3. **लहान भाषा मॉडेल तैनात करणे** - व्यावहारिक तैनाती धोरणे अंमलात आणणे  
4. **मॉडेल स्वरूप रूपांतरण आणि क्वांटायझेशन** - OpenVINO सह 6 फ्रेमवर्कसह प्रगत ऑप्टिमायझेशन  
5. **SLMOps - लहान भाषा मॉडेल ऑपरेशन्स** - उत्पादन जीवनचक्र व्यवस्थापन आणि तैनात करणे  
6. **SLM एजंटिक सिस्टीम्स** - AI एजंट्स, फंक्शन कॉलिंग आणि मॉडेल संदर्भ प्रोटोकॉल  
7. **EdgeAI अंमलबजावणी नमुने** - AI टूलकिट, Windows विकास आणि प्लॅटफॉर्म-विशिष्ट अंमलबजावणी  
8. **Microsoft Foundry Local – संपूर्ण विकासक टूलकिट** - हायब्रिड Azure एकत्रीकरणासह स्थानिक-प्रथम विकास (मॉड्यूल 08)

## अभ्यास मार्गदर्शक कसा वापरायचा

- **प्रगतिशील शिक्षण**: सर्वात सुसंगत शिकण्याचा अनुभव मिळवण्यासाठी मॉड्यूल्स क्रमाने अनुसरण करा  
- **ज्ञान तपासणी**: प्रत्येक विभागानंतर स्व-मूल्यांकन प्रश्नांचा वापर करा  
- **हाताळणीचा सराव**: सैद्धांतिक संकल्पना मजबूत करण्यासाठी सुचवलेले व्यायाम पूर्ण करा  
- **पूरक संसाधने**: आपल्याला सर्वाधिक स्वारस्य असलेल्या विषयांसाठी अतिरिक्त साहित्य एक्सप्लोर करा

## मॉड्यूल 0: EdgeAI परिचय

### मुख्य शिकण्याची उद्दिष्टे

- Edge AI म्हणजे काय आणि आजच्या तंत्रज्ञानाच्या परिदृश्यात त्याचे महत्त्व समजून घ्या  
- Edge AI ने बदललेल्या प्रमुख उद्योगांना आणि त्यांच्या विशिष्ट उपयोग प्रकरणांना ओळखा  
- Edge तैनातीसाठी लहान भाषा मॉडेल्स (SLMs) चे फायदे समजून घ्या  
- संपूर्ण कोर्ससाठी स्पष्ट शिकण्याची अपेक्षा आणि परिणाम स्थापित करा  
- Edge AI क्षेत्रातील करिअर संधी आणि कौशल्य आवश्यकता ओळखा  

### अभ्यास लक्ष केंद्रित क्षेत्रे

#### विभाग 1: Edge AI पॅराडाइम आणि परिभाषा
- **प्राथमिक संकल्पना**:  
  - Edge AI विरुद्ध पारंपरिक क्लाउड AI प्रक्रिया  
  - हार्डवेअर, मॉडेल ऑप्टिमायझेशन आणि व्यवसाय मागण्या यांचे अभिसरण  
  - रिअल-टाइम, गोपनीयता-संरक्षण आणि खर्च-प्रभावी AI तैनात करणे  

#### विभाग 2: उद्योग अनुप्रयोग
- **प्राथमिक संकल्पना**:  
  - उत्पादन आणि उद्योग 4.0: प्रेडिक्टिव मेंटेनन्स आणि गुणवत्ता नियंत्रण  
  - आरोग्यसेवा: निदान इमेजिंग आणि रुग्ण निरीक्षण  
  - स्वायत्त प्रणाली: स्व-चालवणारे वाहन आणि वाहतूक  
  - स्मार्ट शहरे: वाहतूक व्यवस्थापन आणि सार्वजनिक सुरक्षा  
  - ग्राहक तंत्रज्ञान: स्मार्टफोन, वेअरेबल्स आणि स्मार्ट होम्स  

#### विभाग 3: लहान भाषा मॉडेल्स पायाभूत तत्त्वे

- EdgeAI प्रकल्पांसाठी विकास वातावरण सेट करा

  - Edge-विशिष्ट एजंट डिझाइन पॅटर्न
  - एजंट्ससाठी संसाधन ऑप्टिमायझेशन

#### विभाग 2: लहान भाषा मॉडेल्समध्ये फंक्शन कॉलिंग
- **प्राधान्य संकल्पना**:

  - प्लॅटफॉर्म-विशिष्ट अंमलबजावणी (Phi-4-mini, निवडलेले Qwen मॉडेल्स, Microsoft Foundry Local)
  - प्रगत उदाहरणे (मल्टी-एजंट सहयोग, डायनॅमिक टूल निवड)
  - उत्पादन विचार (रेट लिमिटिंग, ऑडिट लॉगिंग, सुरक्षा उपाय)

#### विभाग 3: मॉडेल संदर्भ प्रोटोकॉल (MCP) एकत्रीकरण
- **प्राधान्य संकल्पना**: 
  - प्रोटोकॉल आर्किटेक्चर आणि स्तरित प्रणाली डिझाइन
  - मल्टी-बॅकएंड समर्थन (विकासासाठी Ollama, उत्पादनासाठी vLLM)
  - कनेक्शन प्रोटोकॉल (STDIO आणि SSE मोड)
  - वास्तविक-जगातील अनुप्रयोग (वेब ऑटोमेशन, डेटा प्रक्रिया, API एकत्रीकरण)

## मॉड्यूल 7: EdgeAI अंमलबजावणी नमुने

### मुख्य शिक्षण उद्दिष्टे

- EdgeAI विकास कार्यप्रवाहांसाठी Visual Studio Code साठी AI टूलकिटमध्ये प्रावीण्य मिळवा
- Windows AI Foundry प्लॅटफॉर्म आणि NPU ऑप्टिमायझेशन धोरणांमध्ये तज्ज्ञता मिळवा
- अनेक हार्डवेअर प्लॅटफॉर्म आणि तैनाती परिस्थितींसह EdgeAI अंमलात आणा
- प्लॅटफॉर्म-विशिष्ट ऑप्टिमायझेशनसह उत्पादन-तयार EdgeAI अनुप्रयोग तयार करा

### अभ्यास केंद्रित क्षेत्रे

#### विभाग 1: Visual Studio Code साठी AI टूलकिट
- **प्राधान्य संकल्पना**: 
  - VS Code मध्ये व्यापक Edge AI विकास वातावरण
  - Edge तैनातीसाठी मॉडेल कॅटलॉग आणि शोध
  - स्थानिक चाचणी, ऑप्टिमायझेशन आणि एजंट विकास कार्यप्रवाह
  - Edge परिस्थितींसाठी कार्यप्रदर्शन मॉनिटरिंग आणि मूल्यांकन

#### विभाग 2: Windows EdgeAI विकास मार्गदर्शक
- **प्राधान्य संकल्पना**: 
  - Windows AI Foundry प्लॅटफॉर्मची व्यापक ओळख
  - Phi Silica API कार्यक्षम NPU इनफरन्ससाठी
  - इमेज प्रोसेसिंग आणि OCR साठी Computer Vision APIs
  - स्थानिक विकास आणि चाचणीसाठी Foundry Local CLI

#### विभाग 3: प्लॅटफॉर्म-विशिष्ट अंमलबजावणी
- **प्राधान्य संकल्पना**: 
  - NVIDIA Jetson Orin Nano तैनाती (67 TOPS AI कार्यक्षमता)
  - .NET MAUI आणि ONNX Runtime GenAI सह मोबाइल अनुप्रयोग
  - क्लाउड-Edge हायब्रिड आर्किटेक्चरसह Azure EdgeAI उपाय
  - सार्वत्रिक हार्डवेअर समर्थनासह Windows ML ऑप्टिमायझेशन
  - गोपनीयता-केंद्रित RAG अंमलबजावणीसह Foundry Local अनुप्रयोग

### स्व-मूल्यांकन प्रश्न

1. AI टूलकिट EdgeAI विकास कार्यप्रवाह कसा सुलभ करतो?
2. वेगवेगळ्या हार्डवेअर प्लॅटफॉर्मवर तैनाती धोरणांची तुलना करा.
3. Edge विकासासाठी Windows AI Foundry चे फायदे काय आहेत?
4. आधुनिक Edge AI अनुप्रयोगांमध्ये NPU ऑप्टिमायझेशनची भूमिका स्पष्ट करा.
5. Phi Silica API NPU हार्डवेअरचा वापर कार्यक्षमता सुधारण्यासाठी कसा करते?
6. गोपनीयता-संवेदनशील अनुप्रयोगांसाठी स्थानिक विरुद्ध क्लाउड तैनातीचे फायदे तुलना करा.

### प्रॅक्टिकल सराव

1. **AI टूलकिट सेटअप**: AI टूलकिट कॉन्फिगर करा आणि मॉडेल ऑप्टिमाइझ करा (1 तास)
2. **Windows AI Foundry**: Phi Silica API वापरून सोपा Windows AI अनुप्रयोग तयार करा (1 तास)
3. **क्रॉस-प्लॅटफॉर्म तैनाती**: समान मॉडेल दोन वेगवेगळ्या प्लॅटफॉर्मवर तैनात करा (1 तास)
4. **NPU ऑप्टिमायझेशन**: Windows AI Foundry साधनांसह NPU कार्यप्रदर्शन चाचणी करा (30 मिनिटे)

## मिनी प्रोजेक्ट कल्पना

EdgeAI संकल्पना सराव करण्यासाठी या प्रकल्पांपैकी एक पूर्ण करण्याचा विचार करा (प्रत्येक 2-4 तासांसाठी डिझाइन केलेले):

### नवशिक्या प्रकल्प (2-3 तास प्रत्येक)
1. **एज टेक्स्ट असिस्टंट**: लहान भाषा मॉडेल वापरून एक साधे ऑफलाइन टेक्स्ट पूर्णता साधन तयार करा
2. **मॉडेल तुलना डॅशबोर्ड**: विविध SLM च्या कार्यप्रदर्शन मेट्रिक्सचे मूलभूत व्हिज्युअलायझेशन तयार करा
3. **ऑप्टिमायझेशन प्रयोग**: समान बेस मॉडेलवर विविध क्वांटायझेशन स्तरांचा प्रभाव मोजा

### मध्यम प्रकल्प (3-4 तास प्रत्येक)
4. **AI टूलकिट वर्कफ्लो**: VS Code AI टूलकिट वापरून मॉडेल ऑप्टिमाइझ आणि तैनात करा
5. **Windows AI Foundry अॅप्लिकेशन**: Phi Silica API आणि NPU ऑप्टिमायझेशन वापरून Windows अॅप तयार करा
6. **क्रॉस-प्लॅटफॉर्म तैनाती**: Windows (OpenVINO) आणि मोबाइल (.NET MAUI) वर समान ऑप्टिमाइझ मॉडेल तैनात करा
7. **फंक्शन कॉलिंग एजंट**: एज परिस्थितीसाठी फंक्शन कॉलिंग क्षमता असलेला AI एजंट तयार करा

### प्रगत एकत्रीकरण प्रकल्प (4-5 तास प्रत्येक)
8. **OpenVINO ऑप्टिमायझेशन पाइपलाइन**: NNCF आणि GenAI टूलकिट वापरून संपूर्ण मॉडेल ऑप्टिमायझेशन अंमलात आणा
9. **SLMOps पाइपलाइन**: प्रशिक्षणापासून एज तैनातीपर्यंत संपूर्ण मॉडेल जीवनचक्र अंमलात आणा
10. **मल्टी-मॉडेल एज सिस्टम**: एज हार्डवेअरवर एकत्र काम करणारे अनेक विशेष मॉडेल तैनात करा
11. **MCP एकत्रीकरण प्रणाली**: टूल परस्परसंवादासाठी मॉडेल संदर्भ प्रोटोकॉल वापरून एजंटिक प्रणाली तयार करा
