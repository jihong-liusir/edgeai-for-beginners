# मॉड्यूल 08: Microsoft Foundry Local सह प्रत्यक्ष अनुभव - संपूर्ण डेव्हलपर टूलकिट

## आढावा

Microsoft Foundry Local ही एज AI विकासाची पुढील पिढी आहे, जी डेव्हलपरना स्थानिक स्तरावर AI अ‍ॅप्लिकेशन्स तयार, तैनात आणि स्केल करण्यासाठी शक्तिशाली साधने प्रदान करते, तसेच Azure AI Foundry सह अखंड एकत्रीकरण राखते. या मॉड्यूलमध्ये Foundry Local च्या स्थापनेपासून ते प्रगत एजंट विकासापर्यंत सर्वसमावेशक कव्हरेज दिले आहे.

**मुख्य तंत्रज्ञान:**
- Microsoft Foundry Local CLI आणि SDK
- Azure AI Foundry एकत्रीकरण
- ऑन-डिव्हाइस मॉडेल इन्फरन्स
- स्थानिक मॉडेल कॅशिंग आणि ऑप्टिमायझेशन
- एजंट-आधारित आर्किटेक्चर

## मॉड्यूल शिकण्याची उद्दिष्टे

या मॉड्यूल पूर्ण करून, तुम्ही:

- **Foundry Local सेटअपमध्ये प्राविण्य मिळवा**: Windows 11 साठी Foundry Local स्थापित, कॉन्फिगर आणि ऑप्टिमाइझ करा
- **विविध मॉडेल्स तैनात करा**: CLI कमांड्ससह phi, qwen, deepseek, आणि GPT-OSS-20B मॉडेल्स स्थानिक स्तरावर चालवा
- **उत्पादन उपाय तयार करा**: प्रगत प्रॉम्प्ट इंजिनिअरिंग आणि डेटा एकत्रीकरणासह AI अ‍ॅप्लिकेशन्स तयार करा
- **ओपन-सोर्स इकोसिस्टमचा लाभ घ्या**: Hugging Face मॉडेल्स आणि समुदाय-चालित योगदान एकत्रित करा
- **AI आर्किटेक्चरची तुलना करा**: LLMs आणि SLMs च्या व्यापार-offs आणि तैनाती धोरणे समजून घ्या
- **AI एजंट्स विकसित करा**: Foundry Local च्या आर्किटेक्चर आणि ग्राउंडिंग क्षमतांचा वापर करून बुद्धिमान एजंट्स तयार करा
- **मॉडेल्स साधन म्हणून अंमलात आणा**: एंटरप्राइझ अ‍ॅप्लिकेशन्ससाठी मॉड्यूलर, सानुकूलित AI उपाय तयार करा

## पूर्वअट

### सिस्टम आवश्यकता
- **ऑपरेटिंग सिस्टम**: Windows 11 (22H2 किंवा नंतरचे)
- **मेमरी**: 16GB RAM (मोठ्या मॉडेल्ससाठी 32GB शिफारस केलेले)
- **स्टोरेज**: मॉडेल कॅशिंगसाठी 50GB मोकळी जागा
- **हार्डवेअर**: NPU-सक्षम डिव्हाइस शिफारस केलेले (Copilot+ PC), GPU पर्यायी
- **नेटवर्क**: प्रारंभिक मॉडेल डाउनलोडसाठी उच्च-गती इंटरनेट

### विकास वातावरण
- Visual Studio Code AI Toolkit विस्तारासह
- Python 3.10+ आणि pip
- Git आवृत्ती नियंत्रणासाठी
- PowerShell किंवा कमांड प्रॉम्प्ट
- Azure CLI (क्लाउड एकत्रीकरणासाठी पर्यायी)

### ज्ञान पूर्वअट
- AI/ML संकल्पनांची मूलभूत समज
- कमांड लाइन परिचितता
- Python प्रोग्रामिंग मूलभूत
- REST API संकल्पना
- प्रॉम्प्टिंग आणि मॉडेल इन्फरन्सचे मूलभूत ज्ञान

## शिकण्याचे परिणाम

हे मॉड्यूल पूर्ण केल्यानंतर, तुम्ही सुसज्ज असाल:

### तांत्रिक प्राविण्य
- **तैनात आणि व्यवस्थापित करा**: विकास आणि उत्पादन वातावरणांमध्ये Foundry Local स्थापना
- **मॉडेल्स एकत्रित करा**: Microsoft, Hugging Face, आणि समुदाय स्रोतांमधील विविध मॉडेल्ससह अखंडपणे कार्य करा
- **अ‍ॅप्लिकेशन्स तयार करा**: प्रगत वैशिष्ट्ये आणि ऑप्टिमायझेशनसह उत्पादन-तयार AI अ‍ॅप्लिकेशन्स तयार करा
- **एजंट्स विकसित करा**: ग्राउंडिंग, विचार, आणि साधन एकत्रीकरणासह प्रगत AI एजंट्स अंमलात आणा

### धोरणात्मक समज
- **आर्किटेक्चर निर्णय**: स्थानिक आणि क्लाउड तैनाती दरम्यान माहितीपूर्ण निवडी करा
- **कार्यक्षमता ऑप्टिमायझेशन**: विविध हार्डवेअर कॉन्फिगरेशन्समध्ये इन्फरन्स कार्यक्षमता ऑप्टिमाइझ करा
- **एंटरप्राइझ स्केलिंग**: स्थानिक प्रोटोटाइप्सपासून एंटरप्राइझ तैनातीपर्यंत स्केल होणारी अ‍ॅप्लिकेशन्स डिझाइन करा
- **गोपनीयता आणि सुरक्षा**: स्थानिक इन्फरन्ससह गोपनीयता-संरक्षण AI उपाय अंमलात आणा

### नवकल्पना क्षमता
- **जलद प्रोटोटायपिंग**: AI अ‍ॅप्लिकेशन संकल्पना जलद तयार करा आणि चाचणी करा
- **समुदाय एकत्रीकरण**: ओपन-सोर्स मॉडेल्सचा लाभ घ्या आणि इकोसिस्टममध्ये योगदान द्या
- **प्रगत पॅटर्न्स**: RAG, एजंट्स, आणि साधन एकत्रीकरण यासह अत्याधुनिक AI पॅटर्न्स अंमलात आणा
- **भविष्यातील विकासासाठी तयार**: उदयोन्मुख AI तंत्रज्ञान आणि पॅटर्न्ससाठी तयार अ‍ॅप्लिकेशन्स तयार करा

## सुरुवात करा

1. **तुमचे वातावरण तयार करा**: Windows 11 शिफारस केलेल्या हार्डवेअर वैशिष्ट्यांसह सुनिश्चित करा
2. **पूर्वअटी स्थापित करा**: विकास साधने आणि अवलंबित्वे सेट करा
3. **सत्र 1 पासून प्रारंभ करा**: Foundry Local स्थापना आणि मूलभूत सेटअपसह प्रारंभ करा
4. **क्रमाने प्रगती करा**: सर्वोत्तम शिकण्याच्या प्रगतीसाठी सत्रे क्रमाने पूर्ण करा
5. **सतत सराव करा**: प्रत्यक्ष सराव आणि प्रकल्पांद्वारे संकल्पना लागू करा

## नमुन्यांसाठी जलद प्रारंभ

### 1) वातावरण सेटअप (Windows cmd.exe)
```cmd
cd Module08
py -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

### 2) स्थानिक मॉडेल प्रारंभ करा (नवीन टर्मिनल)
```cmd
foundry model list
foundry model run phi-4-mini
```

### 3) Chainlit डेमो चालवा (सत्र 4)
```cmd
cd Module08
.\.venv\Scripts\activate
chainlit run samples\04\app.py -w
```

### 4) मल्टी-एजंट समन्वयक चालवा (सत्र 5)
```cmd
cd Module08
.\.venv\Scripts\activate
python -m samples.05.agents.coordinator
```

जर तुम्हाला कनेक्शन त्रुटी दिसल्या, तर Foundry Local सत्यापित करा:
```cmd
curl http://localhost:8000/v1/models
```

### राउटर कॉन्फिगरेशन (सत्र 6)
राउटर जलद आरोग्य तपासणी करतो आणि env-आधारित कॉन्फिगरेशनला समर्थन देतो:
```cmd
set BASE_URL=http://localhost:8000
set GENERAL_MODEL=phi-4-mini
set REASONING_MODEL=deepseek-r1-distill-qwen-7b
set CODE_MODEL=qwen2.5-7b-instruct
REM Or provide a full JSON registry
set TOOL_REGISTRY={"general":{"model":"phi-4-mini"}}
python samples\06\router.py "Pick the best model for code refactoring"
```
