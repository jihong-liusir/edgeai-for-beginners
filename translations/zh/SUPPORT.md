# 支持

## 如何提交问题并获取帮助  

此项目使用 GitHub Issues 来跟踪漏洞和功能请求。在提交新问题之前，请先搜索现有问题以避免重复。对于新问题，请将您的漏洞或功能请求提交为一个新的 Issue。

如果需要帮助或对使用此项目有疑问，请：
- 查看 [Azure AI Foundry Discord](https://discord.com/invite/ByRwuEEgH4)，这里汇聚了专家和开发者。
