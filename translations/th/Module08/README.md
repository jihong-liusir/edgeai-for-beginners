# Module 08: การใช้งาน Microsoft Foundry Local - เครื่องมือสำหรับนักพัฒนาครบวงจร

## ภาพรวม

Microsoft Foundry Local เป็นเทคโนโลยีรุ่นใหม่สำหรับการพัฒนา AI บน Edge ที่มอบเครื่องมือทรงพลังให้กับนักพัฒนาในการสร้าง, ใช้งาน, และขยายแอปพลิเคชัน AI ในพื้นที่ พร้อมการผสานรวมกับ Azure AI Foundry อย่างไร้รอยต่อ โมดูลนี้ครอบคลุมทุกขั้นตอนของ Foundry Local ตั้งแต่การติดตั้งไปจนถึงการพัฒนาเอเจนต์ขั้นสูง

**เทคโนโลยีสำคัญ:**
- Microsoft Foundry Local CLI และ SDK
- การผสานรวมกับ Azure AI Foundry
- การประมวลผลโมเดลบนอุปกรณ์
- การแคชและปรับแต่งโมเดลในพื้นที่
- สถาปัตยกรรมแบบเอเจนต์

## วัตถุประสงค์การเรียนรู้

เมื่อจบโมดูลนี้ คุณจะสามารถ:

- **เชี่ยวชาญ Foundry Local**: ติดตั้ง, ตั้งค่า, และปรับแต่งสำหรับการพัฒนาบน Windows 11
- **ใช้งานโมเดลหลากหลาย**: รันโมเดล phi, qwen, deepseek, และ GPT ในพื้นที่ด้วยคำสั่ง CLI
- **สร้างโซลูชันระดับการผลิต**: สร้างแอปพลิเคชัน AI ด้วยการออกแบบ prompt ขั้นสูงและการผสานข้อมูล
- **ใช้ประโยชน์จากระบบโอเพ่นซอร์ส**: ผสานรวมโมเดล Hugging Face และการสนับสนุนจากชุมชน
- **พัฒนา AI Agents**: สร้างเอเจนต์อัจฉริยะด้วยความสามารถในการ grounding และ orchestration
- **นำรูปแบบองค์กรไปใช้**: สร้างโซลูชัน AI ที่มีความเป็นโมดูลและขยายได้สำหรับการใช้งานในระดับการผลิต

## ข้อกำหนดเบื้องต้น

### ความต้องการของระบบ
- **ระบบปฏิบัติการ**: Windows 11 (22H2 หรือใหม่กว่า)
- **หน่วยความจำ**: RAM 16GB (แนะนำ 32GB สำหรับโมเดลขนาดใหญ่)
- **พื้นที่จัดเก็บ**: พื้นที่ว่าง 50GB สำหรับการแคชโมเดล
- **ฮาร์ดแวร์**: อุปกรณ์ที่รองรับ NPU (Copilot+ PC), GPU เป็นตัวเลือก
- **เครือข่าย**: อินเทอร์เน็ตความเร็วสูงสำหรับการดาวน์โหลดโมเดลครั้งแรก

### สภาพแวดล้อมการพัฒนา
- Visual Studio Code พร้อมส่วนขยาย AI Toolkit
- Python 3.10+ และ pip
- Git สำหรับการควบคุมเวอร์ชัน
- PowerShell หรือ Command Prompt
- Azure CLI (ตัวเลือกสำหรับการผสานคลาวด์)

### ความรู้เบื้องต้น
- ความเข้าใจพื้นฐานเกี่ยวกับแนวคิด AI/ML
- ความคุ้นเคยกับการใช้งาน Command Line
- พื้นฐานการเขียนโปรแกรม Python
- แนวคิด REST API
- ความรู้พื้นฐานเกี่ยวกับการออกแบบ prompt และการประมวลผลโมเดล

## ผลลัพธ์การเรียนรู้

เมื่อจบโมดูลนี้ คุณจะมีความสามารถในการ:

### ความเชี่ยวชาญทางเทคนิค
- **ติดตั้งและจัดการ**: การติดตั้ง Foundry Local ในสภาพแวดล้อมการพัฒนาและการผลิต
- **ผสานรวมโมเดล**: ใช้งานโมเดลหลากหลายจาก Microsoft, Hugging Face, และแหล่งชุมชน
- **สร้างแอปพลิเคชัน**: สร้างแอปพลิเคชัน AI พร้อมฟีเจอร์ขั้นสูงและการปรับแต่ง
- **พัฒนาเอเจนต์**: สร้างเอเจนต์ AI ที่ซับซ้อนพร้อมความสามารถในการ grounding, reasoning, และการผสานเครื่องมือ

### ความเข้าใจเชิงกลยุทธ์
- **การตัดสินใจด้านสถาปัตยกรรม**: เลือกใช้งานระหว่างการประมวลผลในพื้นที่กับคลาวด์
- **การปรับแต่งประสิทธิภาพ**: ปรับแต่งประสิทธิภาพการประมวลผลบนฮาร์ดแวร์ที่แตกต่างกัน
- **การขยายในระดับองค์กร**: ออกแบบแอปพลิเคชันที่ขยายได้จากต้นแบบในพื้นที่ไปจนถึงการใช้งานในองค์กร
- **ความเป็นส่วนตัวและความปลอดภัย**: ใช้งานโซลูชัน AI ที่รักษาความเป็นส่วนตัวด้วยการประมวลผลในพื้นที่

### ความสามารถด้านนวัตกรรม
- **การสร้างต้นแบบอย่างรวดเร็ว**: สร้างและทดสอบแนวคิดแอปพลิเคชัน AI อย่างรวดเร็วผ่านรูปแบบตัวอย่างทั้ง 10 แบบ
- **การผสานชุมชน**: ใช้ประโยชน์จากโมเดลโอเพ่นซอร์สและสนับสนุนชุมชน
- **รูปแบบขั้นสูง**: ใช้งานรูปแบบ AI ล้ำสมัย เช่น RAG, เอเจนต์, และการผสานเครื่องมือ
- **ความเชี่ยวชาญใน Framework**: ผสานรวม LangChain, Semantic Kernel, Chainlit, และ Electron อย่างเชี่ยวชาญ
- **การใช้งานในระดับการผลิต**: ใช้งานโซลูชัน AI ที่ขยายได้จากต้นแบบในพื้นที่ไปจนถึงระบบองค์กร
- **การพัฒนาที่พร้อมสำหรับอนาคต**: สร้างแอปพลิเคชันที่พร้อมสำหรับเทคโนโลยีและรูปแบบ AI ที่เกิดขึ้นใหม่

- [ ] ผสาน REST API (ตัวอย่าง 01) สำเร็จ
- [ ] ใช้งาน OpenAI SDK (ตัวอย่าง 02) สำเร็จ
- [ ] ดำเนินการค้นหาและวัดประสิทธิภาพโมเดล (ตัวอย่าง 03) สำเร็จ

### ระดับแอปพลิเคชัน (ตัวอย่าง 04-06)
- [ ] ใช้งานและรันโมเดลอย่างน้อย 4 ตระกูล

### ระดับการผสานขั้นสูง (ตัวอย่าง 07-10)

### ตัวชี้วัดความเชี่ยวชาญ
- [ ] รันตัวอย่างทั้ง 10 แบบสำเร็จโดยไม่มีข้อผิดพลาด
- [ ] ปรับแต่งตัวอย่างอย่างน้อย 3 แบบสำหรับกรณีการใช้งานเฉพาะ
- [ ] ใช้งานตัวอย่าง 2+ แบบในสภาพแวดล้อมที่คล้ายการผลิต
- [ ] สนับสนุนการปรับปรุงหรือขยายโค้ดตัวอย่าง
- [ ] ผสานรูปแบบ Foundry Local เข้ากับโครงการส่วนตัวหรือมืออาชีพ

### การตั้งค่าสภาพแวดล้อม (จำเป็นสำหรับทุกตัว
โมดูลนี้เป็นตัวแทนของความก้าวหน้าล่าสุดในด้านการพัฒนา AI บน Edge โดยผสมผสานเครื่องมือระดับองค์กรของ Microsoft เข้ากับความยืดหยุ่นและนวัตกรรมของระบบโอเพ่นซอร์ส เมื่อคุณเชี่ยวชาญ Foundry Local ผ่านตัวอย่างทั้ง 10 ตัวอย่างที่ครอบคลุม คุณจะอยู่ในตำแหน่งแนวหน้าของการพัฒนาแอปพลิเคชัน AI

**เส้นทางการเรียนรู้ครบถ้วน:**
- **พื้นฐาน** (ตัวอย่าง 01-03): การผสาน API และการจัดการโมเดล
- **แอปพลิเคชัน** (ตัวอย่าง 04-06): RAG, เอเจนต์ และการจัดการเส้นทางอัจฉริยะ
- **ขั้นสูง** (ตัวอย่าง 07-10): เฟรมเวิร์คสำหรับการผลิตและการผสานรวมระดับองค์กร
