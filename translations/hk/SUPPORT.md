# 支援

## 如何提交問題及尋求協助  

此項目使用 GitHub Issues 來追蹤漏洞及功能需求。在提交新問題之前，請先搜尋現有的問題以避免重複。若有新問題，請將您的漏洞或功能需求提交為一個新的 Issue。

如需使用此項目時的協助及問題解答，請：
- 瀏覽 [Azure AI Foundry Discord](https://discord.com/invite/ByRwuEEgH4)，這裡聚集了專家及其他開發者
