# ماڈیول 08: مائیکروسافٹ فاؤنڈری لوکل کے ساتھ عملی تجربہ - مکمل ڈویلپر ٹول کٹ

## جائزہ

مائیکروسافٹ فاؤنڈری لوکل ایج اے آئی ڈیولپمنٹ کی اگلی نسل کی نمائندگی کرتا ہے، جو ڈویلپرز کو طاقتور ٹولز فراہم کرتا ہے تاکہ وہ مقامی طور پر اے آئی ایپلیکیشنز بنا سکیں، تعینات کر سکیں، اور اسکیل کر سکیں، جبکہ Azure AI Foundry کے ساتھ ہموار انضمام کو برقرار رکھ سکیں۔ یہ ماڈیول فاؤنڈری لوکل کی مکمل کوریج فراہم کرتا ہے، انسٹالیشن سے لے کر ایڈوانسڈ ایجنٹ ڈیولپمنٹ تک۔

**اہم ٹیکنالوجیز:**
- مائیکروسافٹ فاؤنڈری لوکل CLI اور SDK
- Azure AI Foundry انضمام
- ڈیوائس پر ماڈل انفرنس
- مقامی ماڈل کیشنگ اور آپٹیمائزیشن
- ایجنٹ پر مبنی آرکیٹیکچرز

## ماڈیول کے سیکھنے کے مقاصد

اس ماڈیول کو مکمل کرنے کے بعد، آپ:

- **فاؤنڈری لوکل سیٹ اپ میں مہارت حاصل کریں گے**: ونڈوز 11 پر فاؤنڈری لوکل کو انسٹال، کنفیگر، اور آپٹیمائز کریں گے۔
- **مختلف ماڈلز کو تعینات کریں گے**: CLI کمانڈز کے ذریعے phi، qwen، deepseek، اور GPT-OSS-20B ماڈلز کو مقامی طور پر چلائیں گے۔
- **پروڈکشن سلوشنز بنائیں گے**: ایڈوانسڈ پرامپٹ انجینئرنگ اور ڈیٹا انٹیگریشن کے ساتھ اے آئی ایپلیکیشنز تیار کریں گے۔
- **اوپن سورس ایکو سسٹم کا فائدہ اٹھائیں گے**: Hugging Face ماڈلز اور کمیونٹی کے اضافے کو انٹیگریٹ کریں گے۔
- **اے آئی آرکیٹیکچرز کا موازنہ کریں گے**: LLMs اور SLMs کے فوائد و نقصانات اور تعیناتی حکمت عملیوں کو سمجھیں گے۔
- **اے آئی ایجنٹس تیار کریں گے**: فاؤنڈری لوکل کی آرکیٹیکچر اور گراؤنڈنگ صلاحیتوں کا استعمال کرتے ہوئے ذہین ایجنٹس بنائیں گے۔
- **ماڈلز کو ٹولز کے طور پر نافذ کریں گے**: انٹرپرائز ایپلیکیشنز کے لیے ماڈیولر، حسب ضرورت اے آئی سلوشنز بنائیں گے۔

## پیشگی شرائط

### سسٹم کی ضروریات
- **آپریٹنگ سسٹم**: ونڈوز 11 (22H2 یا بعد کا ورژن)
- **میموری**: 16GB RAM (بڑے ماڈلز کے لیے 32GB تجویز کردہ)
- **اسٹوریج**: ماڈل کیشنگ کے لیے 50GB خالی جگہ
- **ہارڈویئر**: NPU سے لیس ڈیوائس تجویز کردہ (Copilot+ PC)، GPU اختیاری
- **نیٹ ورک**: ابتدائی ماڈل ڈاؤن لوڈ کے لیے ہائی اسپیڈ انٹرنیٹ

### ڈیولپمنٹ ماحول
- Visual Studio Code کے ساتھ AI Toolkit ایکسٹینشن
- Python 3.10+ اور pip
- ورژن کنٹرول کے لیے Git
- PowerShell یا Command Prompt
- Azure CLI (کلاؤڈ انضمام کے لیے اختیاری)

### علم کی پیشگی شرائط
- اے آئی/ایم ایل کے بنیادی تصورات کی سمجھ
- کمانڈ لائن سے واقفیت
- Python پروگرامنگ کی بنیادی معلومات
- REST API کے تصورات
- پرامپٹنگ اور ماڈل انفرنس کی بنیادی معلومات

## سیکھنے کے نتائج

یہ ماڈیول مکمل کرنے کے بعد، آپ درج ذیل صلاحیتوں سے لیس ہوں گے:

### تکنیکی مہارت
- **تعیناتی اور انتظام**: ترقی اور پروڈکشن ماحول میں فاؤنڈری لوکل انسٹالیشنز کا انتظام کریں۔
- **ماڈلز کا انضمام**: مائیکروسافٹ، Hugging Face، اور کمیونٹی ذرائع سے مختلف ماڈل فیملیز کے ساتھ کام کریں۔
- **ایپلیکیشنز بنائیں**: ایڈوانسڈ فیچرز اور آپٹیمائزیشن کے ساتھ پروڈکشن کے لیے تیار اے آئی ایپلیکیشنز تیار کریں۔
- **ایجنٹس تیار کریں**: گراؤنڈنگ، ریزننگ، اور ٹول انضمام کے ساتھ جدید اے آئی ایجنٹس نافذ کریں۔

### حکمت عملی کی سمجھ
- **آرکیٹیکچر کے فیصلے**: مقامی اور کلاؤڈ تعیناتی کے درمیان باخبر انتخاب کریں۔
- **کارکردگی کی آپٹیمائزیشن**: مختلف ہارڈویئر کنفیگریشنز میں انفرنس کی کارکردگی کو بہتر بنائیں۔
- **انٹرپرائز اسکیلنگ**: مقامی پروٹوٹائپس سے انٹرپرائز تعیناتیوں تک ایپلیکیشنز ڈیزائن کریں۔
- **پرائیویسی اور سیکیورٹی**: مقامی انفرنس کے ساتھ پرائیویسی کو محفوظ رکھنے والے اے آئی سلوشنز نافذ کریں۔

### اختراعی صلاحیتیں
- **تیز رفتار پروٹو ٹائپنگ**: اے آئی ایپلیکیشن کے تصورات کو جلدی سے بنائیں اور جانچیں۔
- **کمیونٹی انضمام**: اوپن سورس ماڈلز کا فائدہ اٹھائیں اور ایکو سسٹم میں تعاون کریں۔
- **ایڈوانسڈ پیٹرنز**: جدید اے آئی پیٹرنز جیسے RAG، ایجنٹس، اور ٹول انضمام کو نافذ کریں۔
- **مستقبل کے لیے تیار ترقی**: ابھرتی ہوئی اے آئی ٹیکنالوجیز اور پیٹرنز کے لیے تیار ایپلیکیشنز بنائیں۔

## نمونوں کے لیے فوری آغاز

### 1) ماحول سیٹ اپ (Windows cmd.exe)
```cmd
cd Module08
py -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

### 2) ایک مقامی ماڈل شروع کریں (نیا ٹرمینل)
```cmd
foundry model list
foundry model run phi-4-mini
```

### 3) Chainlit ڈیمو چلائیں (سیشن 4)
```cmd
cd Module08
.\.venv\Scripts\activate
chainlit run samples\04\app.py -w
```

### 4) ملٹی ایجنٹ کوآرڈینیٹر چلائیں (سیشن 5)
```cmd
cd Module08
.\.venv\Scripts\activate
python -m samples.05.agents.coordinator
```

اگر آپ کو کنکشن کی غلطیاں نظر آئیں، تو فاؤنڈری لوکل کی توثیق کریں:
```cmd
curl http://localhost:8000/v1/models
```

### روٹر کنفیگریشن (سیشن 6)
روٹر ایک فوری صحت کی جانچ کرتا ہے اور ماحول پر مبنی کنفیگریشن کو سپورٹ کرتا ہے:
```cmd
set BASE_URL=http://localhost:8000
set GENERAL_MODEL=phi-4-mini
set REASONING_MODEL=deepseek-r1-distill-qwen-7b
set CODE_MODEL=qwen2.5-7b-instruct
REM Or provide a full JSON registry
set TOOL_REGISTRY={"general":{"model":"phi-4-mini"}}
python samples\06\router.py "Pick the best model for code refactoring"
```
