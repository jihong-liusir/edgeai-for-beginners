# Module 08: Microsoft Foundry Local - সম্পূর্ণ ডেভেলপার টুলকিটের হাতে-কলমে অভিজ্ঞতা

## সংক্ষিপ্ত বিবরণ

Microsoft Foundry Local হলো এজ AI ডেভেলপমেন্টের পরবর্তী প্রজন্ম, যা ডেভেলপারদের শক্তিশালী টুল সরবরাহ করে AI অ্যাপ্লিকেশন স্থানীয়ভাবে তৈরি, ডিপ্লয় এবং স্কেল করার জন্য, এবং একই সাথে Azure AI Foundry-এর সাথে নির্বিঘ্ন ইন্টিগ্রেশন বজায় রাখে। এই মডিউলটি Foundry Local-এর ইনস্টলেশন থেকে শুরু করে উন্নত এজেন্ট ডেভেলপমেন্ট পর্যন্ত বিস্তৃত কভারেজ প্রদান করে।

**মূল প্রযুক্তি:**
- Microsoft Foundry Local CLI এবং SDK
- Azure AI Foundry ইন্টিগ্রেশন
- ডিভাইসে মডেল ইনফারেন্স
- স্থানীয় মডেল ক্যাশিং এবং অপ্টিমাইজেশন
- এজেন্ট-ভিত্তিক আর্কিটেকচার

## মডিউল শেখার লক্ষ্যসমূহ

এই মডিউল সম্পন্ন করার মাধ্যমে আপনি:

- **Foundry Local সেটআপে দক্ষতা অর্জন করবেন**: Windows 11 ডেভেলপমেন্টের জন্য Foundry Local ইনস্টল, কনফিগার এবং অপ্টিমাইজ করবেন
- **বিভিন্ন মডেল ডিপ্লয় করবেন**: CLI কমান্ড ব্যবহার করে phi, qwen, deepseek এবং GPT-OSS-20B মডেল স্থানীয়ভাবে চালাবেন
- **প্রোডাকশন সমাধান তৈরি করবেন**: উন্নত প্রম্পট ইঞ্জিনিয়ারিং এবং ডেটা ইন্টিগ্রেশনের মাধ্যমে AI অ্যাপ্লিকেশন তৈরি করবেন
- **ওপেন-সোর্স ইকোসিস্টেম ব্যবহার করবেন**: Hugging Face মডেল এবং কমিউনিটি-চালিত সংযোজন ইন্টিগ্রেট করবেন
- **AI আর্কিটেকচার তুলনা করবেন**: LLMs বনাম SLMs-এর ট্রেড-অফ এবং ডিপ্লয়মেন্ট কৌশল বুঝবেন
- **AI এজেন্ট তৈরি করবেন**: Foundry Local-এর আর্কিটেকচার এবং গ্রাউন্ডিং ক্ষমতা ব্যবহার করে বুদ্ধিমান এজেন্ট তৈরি করবেন
- **মডেলকে টুল হিসেবে প্রয়োগ করবেন**: এন্টারপ্রাইজ অ্যাপ্লিকেশনের জন্য মডুলার, কাস্টমাইজযোগ্য AI সমাধান তৈরি করবেন

### [5: Foundry Local দিয়ে দ্রুত AI-চালিত এজেন্ট তৈরি করা](./05.AIPoweredAgents.md)
**ফোকাস**: দ্রুত GenAI অ্যাপ ডেভেলপমেন্ট, সিস্টেম প্রম্পট, গ্রাউন্ডিং এবং এজেন্ট আর্কিটেকচার

**আপনি যা শিখবেন:**
- Foundry Local এজেন্ট আর্কিটেকচার এবং ডিজাইন প্যাটার্ন
- এজেন্ট আচরণের জন্য সিস্টেম প্রম্পট ইঞ্জিনিয়ারিং
- নির্ভরযোগ্য এজেন্ট প্রতিক্রিয়ার জন্য গ্রাউন্ডিং কৌশল
- দ্রুত GenAI অ্যাপ্লিকেশন ডেভেলপমেন্ট ওয়ার্কফ্লো
- এজেন্ট অর্কেস্ট্রেশন এবং মাল্টি-এজেন্ট সিস্টেম
- প্রোডাকশন ডিপ্লয়মেন্ট কৌশল AI এজেন্টের জন্য

**সময়কাল**: ৩-৪ ঘণ্টা  
**প্রয়োজনীয়তা**: সেশন ১-৪ সম্পন্ন, AI এজেন্টের মৌলিক ধারণা

### [6: Foundry Local - মডেলকে টুল হিসেবে ব্যবহার করা](./06.ModelsAsTools.md)

**আপনি যা শিখবেন:**
- AI মডেলকে মডুলার, কাস্টমাইজযোগ্য টুল হিসেবে ব্যবহার করা
- ক্লাউড নির্ভরতা ছাড়াই ডিভাইসে ডিপ্লয়মেন্ট
- লো-লেটেন্সি, খরচ-সাশ্রয়ী এবং গোপনীয়তা-সংরক্ষণকারী ইনফারেন্স
- SDK, API এবং CLI ইন্টিগ্রেশন প্যাটার্ন
- নির্দিষ্ট ব্যবহার কেসের জন্য মডেল কাস্টমাইজেশন
- স্থানীয় থেকে Azure AI Foundry পর্যন্ত স্কেলিং কৌশল
- এন্টারপ্রাইজ-প্রস্তুত AI অ্যাপ্লিকেশন আর্কিটেকচার

**সময়কাল**: ৩-৪ ঘণ্টা  
**প্রয়োজনীয়তা**: পূর্ববর্তী সমস্ত সেশন, এন্টারপ্রাইজ ডেভেলপমেন্ট অভিজ্ঞতা সহায়ক

## প্রয়োজনীয়তা

### সিস্টেমের প্রয়োজনীয়তা
- **অপারেটিং সিস্টেম**: Windows 11 (22H2 বা পরবর্তী)
- **মেমরি**: 16GB RAM (বড় মডেলের জন্য 32GB সুপারিশকৃত)
- **স্টোরেজ**: মডেল ক্যাশিংয়ের জন্য 50GB ফ্রি স্পেস
- **হার্ডওয়্যার**: NPU-সক্ষম ডিভাইস সুপারিশকৃত (Copilot+ PC), GPU ঐচ্ছিক
- **নেটওয়ার্ক**: প্রাথমিক মডেল ডাউনলোডের জন্য উচ্চ-গতির ইন্টারনেট

### ডেভেলপমেন্ট পরিবেশ
- Visual Studio Code AI Toolkit এক্সটেনশন সহ
- Python 3.10+ এবং pip
- Git ভার্সন কন্ট্রোলের জন্য
- PowerShell বা Command Prompt
- Azure CLI (ক্লাউড ইন্টিগ্রেশনের জন্য ঐচ্ছিক)

### জ্ঞানের প্রয়োজনীয়তা
- AI/ML ধারণার মৌলিক বোঝাপড়া
- কমান্ড লাইন পরিচিতি
- Python প্রোগ্রামিংয়ের মৌলিক জ্ঞান
- REST API ধারণা
- প্রম্পটিং এবং মডেল ইনফারেন্সের মৌলিক জ্ঞান

### প্রযুক্তিগত দক্ষতা
- **ডিপ্লয় এবং পরিচালনা**: ডেভেলপমেন্ট এবং প্রোডাকশন পরিবেশে Foundry Local ইনস্টলেশন

- **অ্যাপ্লিকেশন তৈরি**: উন্নত বৈশিষ্ট্য এবং অপ্টিমাইজেশন সহ প্রোডাকশন-প্রস্তুত AI অ্যাপ্লিকেশন তৈরি
- **এজেন্ট ডেভেলপমেন্ট**: গ্রাউন্ডিং, রিজনিং এবং টুল ইন্টিগ্রেশন সহ উন্নত AI এজেন্ট বাস্তবায়ন

### কৌশলগত বোঝাপড়া
- **আর্কিটেকচার সিদ্ধান্ত**: স্থানীয় বনাম ক্লাউড ডিপ্লয়মেন্টের মধ্যে তথ্যপূর্ণ পছন্দ করা
- **পারফরম্যান্স অপ্টিমাইজেশন**: বিভিন্ন হার্ডওয়্যার কনফিগারেশনে ইনফারেন্স পারফরম্যান্স অপ্টিমাইজ করা
- **এন্টারপ্রাইজ স্কেলিং**: স্থানীয় প্রোটোটাইপ থেকে এন্টারপ্রাইজ ডিপ্লয়মেন্টে স্কেল করার জন্য অ্যাপ্লিকেশন ডিজাইন
- **গোপনীয়তা এবং নিরাপত্তা**: স্থানীয় ইনফারেন্স সহ গোপনীয়তা-সংরক্ষণকারী AI সমাধান বাস্তবায়ন

### উদ্ভাবনী সক্ষমতা
- **দ্রুত প্রোটোটাইপিং**: দ্রুত AI অ্যাপ্লিকেশন ধারণা তৈরি এবং পরীক্ষা করা
- **কমিউনিটি ইন্টিগ্রেশন**: ওপেন-সোর্স মডেল ব্যবহার এবং ইকোসিস্টেমে অবদান রাখা
- **উন্নত প্যাটার্ন**: RAG, এজেন্ট এবং টুল ইন্টিগ্রেশন সহ আধুনিক AI প্যাটার্ন বাস্তবায়ন
- **ভবিষ্যৎ-প্রস্তুত ডেভেলপমেন্ট**: উদীয়মান AI প্রযুক্তি এবং প্যাটার্নের জন্য প্রস্তুত অ্যাপ্লিকেশন তৈরি

foundry model list

cd Module08

chainlit run samples\04\app.py -w

### ৪) মাল্টি-এজেন্ট কোঅর্ডিনেটর চালান (সেশন ৫)

cd Module08

```cmd
curl http://localhost:8000/v1/models
```

### রাউটার কনফিগারেশন (সেশন ৬)
রাউটার একটি দ্রুত স্বাস্থ্য পরীক্ষা করে এবং env-ভিত্তিক কনফিগারেশন সমর্থন করে:
```cmd
set BASE_URL=http://localhost:8000
set GENERAL_MODEL=phi-4-mini
set REASONING_MODEL=deepseek-r1-distill-qwen-7b
set CODE_MODEL=qwen2.5-7b-instruct
REM Or provide a full JSON registry
set TOOL_REGISTRY={"general":{"model":"phi-4-mini"}}
python samples\06\router.py "Pick the best model for code refactoring"
```

এই মডিউল এজ AI ডেভেলপমেন্টের সর্বাধুনিক প্রতিনিধিত্ব করে, যা Microsoft-এর এন্টারপ্রাইজ-গ্রেড টুল এবং ওপেন-সোর্স ইকোসিস্টেমের নমনীয়তা ও উদ্ভাবনকে একত্রিত করে। Foundry Local-এ দক্ষতা অর্জনের মাধ্যমে, আপনি AI অ্যাপ্লিকেশন ডেভেলপমেন্টের অগ্রভাগে অবস্থান করবেন।

Azure OpenAI (সেশন ২) এর জন্য, প্রয়োজনীয় পরিবেশ ভেরিয়েবল এবং API সংস্করণ সেটিংসের জন্য নমুনা README দেখুন।
