# Модул 08: Практична примена са Microsoft Foundry Local - Комплетан алат за развој

## Преглед

Microsoft Foundry Local представља следећу генерацију развоја edge AI технологије, пружајући програмерима моћне алате за креирање, имплементацију и скалирање AI апликација локално, уз беспрекорну интеграцију са Azure AI Foundry. Овај модул нуди свеобухватан преглед Foundry Local-а, од инсталације до напредног развоја агената.

**Кључне технологије:**
- Microsoft Foundry Local CLI и SDK
- Интеграција са Azure AI Foundry
- Инференција модела на уређају
- Локално кеширање и оптимизација модела
- Архитектуре засноване на агентима

## Циљеви учења модула

Завршетком овог модула, научићете:

- **Овладавање поставком Foundry Local-а**: Инсталирање, конфигурисање и оптимизација Foundry Local-а за развој на Windows 11
- **Имплементација различитих модела**: Покретање phi, qwen, deepseek и GPT-OSS-20B модела локално помоћу CLI команди
- **Креирање производних решења**: Развој AI апликација уз напредно инжењерство упита и интеграцију података
- **Искоришћавање отвореног кода**: Интеграција Hugging Face модела и доприноса заједнице
- **Поређење AI архитектура**: Разумевање компромиса између LLMs и SLMs и стратегија имплементације
- **Развој AI агената**: Креирање интелигентних агената користећи архитектуру Foundry Local-а и технике за утемељење
- **Имплементација модела као алата**: Креирање модуларних, прилагодљивих AI решења за пословне апликације

## Предуслови

### Системски захтеви
- **Оперативни систем**: Windows 11 (22H2 или новији)
- **Меморија**: 16GB RAM-а (32GB препоручено за веће моделе)
- **Складиште**: 50GB слободног простора за кеширање модела
- **Хардвер**: Уређај са NPU-ом препоручен (Copilot+ PC), GPU опционално
- **Мрежа**: Брзи интернет за почетно преузимање модела

### Развојно окружење
- Visual Studio Code са AI Toolkit екстензијом
- Python 3.10+ и pip
- Git за контролу верзија
- PowerShell или Command Prompt
- Azure CLI (опционално за cloud интеграцију)

### Знање
- Основно разумевање AI/ML концепата
- Познавање командне линије
- Основе Python програмирања
- Концепти REST API-а
- Основно знање о упитима и инференцији модела

## Брзи почетак за примере

### 1) Постављање окружења (Windows cmd.exe)
```cmd
cd Module08
py -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

### 2) Покрените локални модел (нови терминал)
```cmd
foundry model list
foundry model run phi-4-mini
```

### 3) Покрените Chainlit демонстрацију (Сесија 4)
```cmd
cd Module08
.\.venv\Scripts\activate
chainlit run samples\04\app.py -w
```

### 4) Покрените координатора за више агената (Сесија 5)
```cmd
cd Module08
.\.venv\Scripts\activate
python -m samples.05.agents.coordinator
```

Ако видите грешке у вези, проверите Foundry Local:
```cmd
curl http://localhost:8000/v1/models
```

### Конфигурација рутера (Сесија 6)
Рутер врши брзу проверу здравља и подржава конфигурацију засновану на окружењу:
```cmd
set BASE_URL=http://localhost:8000
set GENERAL_MODEL=phi-4-mini
set REASONING_MODEL=deepseek-r1-distill-qwen-7b
set CODE_MODEL=qwen2.5-7b-instruct
REM Or provide a full JSON registry
set TOOL_REGISTRY={"general":{"model":"phi-4-mini"}}
python samples\06\router.py "Pick the best model for code refactoring"
```

Овај модул представља најсавременији развој edge AI технологије, комбинујући Microsoft-ове алате за предузећа са флексибилношћу и иновацијама екосистема отвореног кода. Овладавањем Foundry Local-ом, бићете на челу развоја AI апликација.

За Azure OpenAI (Сесија 2), погледајте README за примере потребних променљивих окружења и подешавања верзије API-а.
