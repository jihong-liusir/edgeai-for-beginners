# Module 08: Microsoft Foundry Local - पूर्ण डेभलपर टूलकिटमा व्यावहारिक अनुभव

## परिचय

Microsoft Foundry Local ले एज AI विकासको नयाँ पुस्ता प्रतिनिधित्व गर्दछ, जसले डेभलपरहरूलाई शक्तिशाली उपकरणहरू प्रदान गर्दछ ताकि उनीहरूले स्थानीय रूपमा AI एप्लिकेसन निर्माण, तैनात, र स्केल गर्न सकून्, Azure AI Foundry सँग सहज एकीकरण कायम राख्दै। यो मोड्युलले Foundry Local को स्थापना देखि लिएर उन्नत एजेन्ट विकाससम्मको व्यापक कभरेज प्रदान गर्दछ।

**मुख्य प्रविधिहरू:**
- Microsoft Foundry Local CLI र SDK
- Azure AI Foundry एकीकरण
- उपकरणमा मोडेल इन्फरेन्स
- स्थानीय मोडेल क्यासिङ र अनुकूलन
- एजेन्ट-आधारित आर्किटेक्चरहरू

## मोड्युल सिकाइ उद्देश्यहरू

यो मोड्युल पूरा गरेपछि, तपाईं:

- **Foundry Local सेटअपमा महारत हासिल गर्नुहोस्**: Windows 11 विकासका लागि Foundry Local स्थापना, कन्फिगर, र अनुकूलन गर्नुहोस्
- **विविध मोडेलहरू तैनात गर्नुहोस्**: phi, qwen, deepseek, र GPT-OSS-20B मोडेलहरू CLI कमाण्डहरू प्रयोग गरी स्थानीय रूपमा चलाउनुहोस्
- **उत्पादन समाधानहरू निर्माण गर्नुहोस्**: उन्नत प्रम्प्ट इन्जिनियरिङ र डाटा एकीकरणको साथ AI एप्लिकेसनहरू सिर्जना गर्नुहोस्
- **ओपन-सोर्स इकोसिस्टमको लाभ उठाउनुहोस्**: Hugging Face मोडेलहरू र समुदाय-चालित थपहरू एकीकृत गर्नुहोस्
- **AI आर्किटेक्चरहरूको तुलना गर्नुहोस्**: LLMs र SLMs बीचको व्यापार-सम्झौता र तैनाती रणनीतिहरू बुझ्नुहोस्
- **AI एजेन्टहरू विकास गर्नुहोस्**: Foundry Local को आर्किटेक्चर र ग्राउन्डिङ क्षमताहरू प्रयोग गरी बुद्धिमान एजेन्टहरू निर्माण गर्नुहोस्
- **मोडेलहरू उपकरणको रूपमा कार्यान्वयन गर्नुहोस्**: उद्यम एप्लिकेसनहरूको लागि मोड्युलर, अनुकूलन योग्य AI समाधानहरू सिर्जना गर्नुहोस्

## पूर्व-आवश्यकताहरू

### प्रणाली आवश्यकताहरू
- **अपरेटिङ सिस्टम**: Windows 11 (22H2 वा पछिल्लो)
- **मेमोरी**: 16GB RAM (ठूला मोडेलहरूको लागि 32GB सिफारिस गरिएको)
- **स्टोरेज**: मोडेल क्यासिङका लागि 50GB खाली ठाउँ
- **हार्डवेयर**: NPU-सक्षम उपकरण सिफारिस गरिएको (Copilot+ PC), GPU वैकल्पिक
- **नेटवर्क**: प्रारम्भिक मोडेल डाउनलोडका लागि उच्च-गति इन्टरनेट

### विकास वातावरण
- Visual Studio Code AI Toolkit एक्सटेन्सनसहित
- Python 3.10+ र pip
- Git संस्करण नियन्त्रणका लागि
- PowerShell वा Command Prompt
- Azure CLI (क्लाउड एकीकरणका लागि वैकल्पिक)

### ज्ञान आवश्यकताहरू
- AI/ML अवधारणाको आधारभूत समझ
- कमाण्ड लाइन परिचितता
- Python प्रोग्रामिङ आधारभूत
- REST API अवधारणाहरू
- प्रम्प्टिङ र मोडेल इन्फरेन्सको आधारभूत ज्ञान

## सिकाइ परिणामहरू

यो मोड्युल पूरा गरेपछि, तपाईं सक्षम हुनुहुनेछ:

### प्राविधिक महारत
- **तैनात र व्यवस्थापन गर्नुहोस्**: विकास र उत्पादन वातावरणहरूमा Foundry Local स्थापना
- **मोडेलहरू एकीकृत गर्नुहोस्**: Microsoft, Hugging Face, र समुदाय स्रोतहरूबाट विविध मोडेल परिवारहरूसँग सहज काम गर्नुहोस्
- **एप्लिकेसनहरू निर्माण गर्नुहोस्**: उन्नत सुविधाहरू र अनुकूलनसहित उत्पादन-तयार AI एप्लिकेसनहरू सिर्जना गर्नुहोस्
- **एजेन्टहरू विकास गर्नुहोस्**: ग्राउन्डिङ, तर्क, र उपकरण एकीकरणसहित परिष्कृत AI एजेन्टहरू कार्यान्वयन गर्नुहोस्

### रणनीतिक समझ
- **आर्किटेक्चर निर्णयहरू**: स्थानीय बनाम क्लाउड तैनाती बीच सूचित विकल्पहरू बनाउनुहोस्
- **प्रदर्शन अनुकूलन**: विभिन्न हार्डवेयर कन्फिगरेसनहरूमा इन्फरेन्स प्रदर्शन अनुकूलन गर्नुहोस्
- **उद्यम स्केलिङ**: स्थानीय प्रोटोटाइपहरूबाट उद्यम तैनातीसम्म स्केल गर्ने एप्लिकेसनहरू डिजाइन गर्नुहोस्
- **गोपनीयता र सुरक्षा**: स्थानीय इन्फरेन्ससहित गोपनीयता-संरक्षण AI समाधानहरू कार्यान्वयन गर्नुहोस्

### नवाचार क्षमता
- **छिटो प्रोटोटाइपिङ**: छिटो AI एप्लिकेसन अवधारणाहरू निर्माण र परीक्षण गर्नुहोस्
- **समुदाय एकीकरण**: ओपन-सोर्स मोडेलहरूको लाभ उठाउनुहोस् र इकोसिस्टममा योगदान गर्नुहोस्
- **उन्नत ढाँचाहरू**: RAG, एजेन्टहरू, र उपकरण एकीकरण सहित अत्याधुनिक AI ढाँचाहरू कार्यान्वयन गर्नुहोस्
- **भविष्य-तयार विकास**: उदीयमान AI प्रविधिहरू र ढाँचाहरूका लागि तयार एप्लिकेसनहरू निर्माण गर्नुहोस्

### राउटर कन्फिगरेसन (सत्र ६)
राउटरले छिटो स्वास्थ्य जाँच गर्दछ र env-आधारित कन्फिगलाई समर्थन गर्दछ:
```cmd
set BASE_URL=http://localhost:8000
set GENERAL_MODEL=phi-4-mini
set REASONING_MODEL=deepseek-r1-distill-qwen-7b
set CODE_MODEL=qwen2.5-7b-instruct
REM Or provide a full JSON registry
set TOOL_REGISTRY={"general":{"model":"phi-4-mini"}}
python samples\06\router.py "Pick the best model for code refactoring"
```

यो मोड्युल एज AI विकासको अत्याधुनिक प्रतिनिधित्व गर्दछ, जसले Microsoft को उद्यम-ग्रेड उपकरणहरूलाई ओपन-सोर्स इकोसिस्टमको लचिलोपन र नवाचारसँग संयोजन गर्दछ। Foundry Local मा महारत हासिल गरेर, तपाईं AI एप्लिकेसन विकासको अग्रभागमा रहन सक्षम हुनुहुनेछ।

Azure OpenAI (सत्र २) को लागि, आवश्यक वातावरण चरहरू र API संस्करण सेटिङहरूको लागि नमूना README हेर्नुहोस्।
