# ਮੋਡਿਊਲ 08: ਮਾਈਕਰੋਸਾਫਟ ਫਾਊਂਡਰੀ ਲੋਕਲ ਨਾਲ ਹੈਂਡਸ-ਆਨ - ਪੂਰਾ ਡਿਵੈਲਪਰ ਟੂਲਕਿਟ

## ਝਲਕ

ਮਾਈਕਰੋਸਾਫਟ ਫਾਊਂਡਰੀ ਲੋਕਲ ਐਜ AI ਵਿਕਾਸ ਦੀ ਅਗਲੀ ਪੀੜੀ ਨੂੰ ਦਰਸਾਉਂਦਾ ਹੈ, ਜੋ ਡਿਵੈਲਪਰਾਂ ਨੂੰ ਤਾਕਤਵਰ ਟੂਲਾਂ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ ਤਾਂ ਜੋ ਉਹ AI ਐਪਲੀਕੇਸ਼ਨ ਨੂੰ ਸਥਾਨਕ ਤੌਰ 'ਤੇ ਬਣਾਉਣ, ਤੈਨਾਤ ਕਰਨ ਅਤੇ ਵਧਾਉਣ ਦੇ ਯੋਗ ਹੋਣ। ਇਹ ਮੋਡਿਊਲ ਫਾਊਂਡਰੀ ਲੋਕਲ ਦੇ ਸਥਾਪਨਾ ਤੋਂ ਲੈ ਕੇ ਉੱਨਤ ਏਜੰਟ ਵਿਕਾਸ ਤੱਕ ਦੀ ਵਿਸਤ੍ਰਿਤ ਕਵਰੇਜ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ।

**ਮੁੱਖ ਤਕਨਾਲੋਜੀਆਂ:**
- ਮਾਈਕਰੋਸਾਫਟ ਫਾਊਂਡਰੀ ਲੋਕਲ CLI ਅਤੇ SDK
- Azure AI Foundry ਇੰਟੀਗ੍ਰੇਸ਼ਨ
- ਡਿਵਾਈਸ 'ਤੇ ਮਾਡਲ ਇੰਫਰੈਂਸ
- ਸਥਾਨਕ ਮਾਡਲ ਕੈਸ਼ਿੰਗ ਅਤੇ ਅਪਟਿਮਾਈਜ਼ੇਸ਼ਨ
- ਏਜੰਟ-ਅਧਾਰਿਤ ਆਰਕੀਟੈਕਚਰ

## ਮੋਡਿਊਲ ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

ਇਸ ਮੋਡਿਊਲ ਨੂੰ ਪੂਰਾ ਕਰਨ ਦੁਆਰਾ, ਤੁਸੀਂ:

- **Foundry Local ਸੈਟਅਪ ਵਿੱਚ ਮਾਹਰ ਹੋਵੋ**: Windows 11 ਵਿਕਾਸ ਲਈ Foundry Local ਨੂੰ ਸਥਾਪਿਤ, ਸੰਰਚਿਤ ਅਤੇ ਅਪਟਿਮਾਈਜ਼ ਕਰੋ
- **ਵਿਭਿੰਨ ਮਾਡਲ ਤੈਨਾਤ ਕਰੋ**: CLI ਕਮਾਂਡਾਂ ਨਾਲ phi, qwen, deepseek, ਅਤੇ GPT-OSS-20B ਮਾਡਲ ਸਥਾਨਕ ਤੌਰ 'ਤੇ ਚਲਾਓ
- **ਪ੍ਰੋਡਕਸ਼ਨ ਹੱਲ ਬਣਾਓ**: ਉੱਨਤ ਪ੍ਰੋਮਪਟ ਇੰਜੀਨੀਅਰਿੰਗ ਅਤੇ ਡਾਟਾ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਨਾਲ AI ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਓ
- **ਖੁੱਲ੍ਹੇ-ਸਰੋਤ ਪੇਸ਼ਕਸ਼ਾਂ ਦਾ ਲਾਭ ਉਠਾਓ**: Hugging Face ਮਾਡਲਾਂ ਅਤੇ ਕਮਿਊਨਿਟੀ-ਚਲਿਤ ਸ਼ਾਮਲੀਆਂ ਨੂੰ ਇੰਟੀਗ੍ਰੇਟ ਕਰੋ
- **AI ਆਰਕੀਟੈਕਚਰਾਂ ਦੀ ਤੁਲਨਾ ਕਰੋ**: LLMs ਅਤੇ SLMs ਦੇ ਫਰਕ ਅਤੇ ਤੈਨਾਤੀ ਰਣਨੀਤੀਆਂ ਨੂੰ ਸਮਝੋ
- **AI ਏਜੰਟ ਵਿਕਸਿਤ ਕਰੋ**: Foundry Local ਦੀ ਆਰਕੀਟੈਕਚਰ ਅਤੇ ਗ੍ਰਾਊਂਡਿੰਗ ਯੋਗਤਾਵਾਂ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਬੁੱਧੀਮਾਨ ਏਜੰਟ ਬਣਾਓ
- **ਮਾਡਲਾਂ ਨੂੰ ਟੂਲ ਵਜੋਂ ਲਾਗੂ ਕਰੋ**: ਕਾਰੋਬਾਰੀ ਐਪਲੀਕੇਸ਼ਨਾਂ ਲਈ ਮਾਡੂਲਰ, ਕਸਟਮਾਈਜ਼ੇਬਲ AI ਹੱਲ ਬਣਾਓ

- ਵਧੀਆ ਪ੍ਰਦਰਸ਼ਨ ਲਈ ਮਾਡਲ ਕੈਸ਼ਿੰਗ ਰਣਨੀਤੀਆਂ
- ਹਾਰਡਵੇਅਰ ਐਕਸਲੇਰੇਸ਼ਨ ਸੈਟਅਪ ਅਤੇ ਅਪਟਿਮਾਈਜ਼ੇਸ਼ਨ
- phi, qwen, deepseek, ਅਤੇ GPT-OSS-20B ਮਾਡਲਾਂ ਦੀ ਹੈਂਡਸ-ਆਨ ਤੈਨਾਤੀ

- ਭਰੋਸੇਯੋਗ ਏਜੰਟ ਜਵਾਬਾਂ ਲਈ ਗ੍ਰਾਊਂਡਿੰਗ ਤਕਨੀਕਾਂ

## ਪੂਰਵ-ਸ਼ਰਤਾਂ

### ਸਿਸਟਮ ਦੀਆਂ ਲੋੜਾਂ
- **ਆਪਰੇਟਿੰਗ ਸਿਸਟਮ**: Windows 11 (22H2 ਜਾਂ ਇਸ ਤੋਂ ਬਾਅਦ)
- **ਮੈਮਰੀ**: 16GB RAM (ਵੱਡੇ ਮਾਡਲਾਂ ਲਈ 32GB ਦੀ ਸਿਫਾਰਸ਼ ਕੀਤੀ ਗਈ)
- **ਸਟੋਰੇਜ**: ਮਾਡਲ ਕੈਸ਼ਿੰਗ ਲਈ 50GB ਖਾਲੀ ਜਗ੍ਹਾ
- **ਹਾਰਡਵੇਅਰ**: NPU-ਯੋਗ ਡਿਵਾਈਸ ਦੀ ਸਿਫਾਰਸ਼ ਕੀਤੀ ਗਈ (Copilot+ PC), GPU ਵਿਕਲਪਿਕ
- **ਨੈੱਟਵਰਕ**: ਸ਼ੁਰੂਆਤੀ ਮਾਡਲ ਡਾਊਨਲੋਡ ਲਈ ਹਾਈ-ਸਪੀਡ ਇੰਟਰਨੈੱਟ

### ਵਿਕਾਸ ਦਾ ਮਾਹੌਲ
- Visual Studio Code AI Toolkit ਐਕਸਟੈਂਸ਼ਨ ਨਾਲ
- Python 3.10+ ਅਤੇ pip
- Git ਵਰਜਨ ਕੰਟਰੋਲ ਲਈ
- PowerShell ਜਾਂ Command Prompt
- Azure CLI (ਕਲਾਉਡ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਲਈ ਵਿਕਲਪਿਕ)

### ਗਿਆਨ ਪੂਰਵ-ਸ਼ਰਤਾਂ
- AI/ML ਸੰਕਲਪਾਂ ਦੀ ਬੁਨਿਆਦੀ ਸਮਝ
- ਕਮਾਂਡ ਲਾਈਨ ਨਾਲ ਜਾਣ-ਪਛਾਣ
- Python ਪ੍ਰੋਗਰਾਮਿੰਗ ਬੁਨਿਆਦੀਆਂ
- REST API ਸੰਕਲਪ
- ਪ੍ਰੋਮਪਟਿੰਗ ਅਤੇ ਮਾਡਲ ਇੰਫਰੈਂਸ ਦਾ ਬੁਨਿਆਦੀ ਗਿਆਨ

## ਸਿੱਖਣ ਦੇ ਨਤੀਜੇ

ਇਸ ਮੋਡਿਊਲ ਨੂੰ ਪੂਰਾ ਕਰਨ ਤੋਂ ਬਾਅਦ, ਤੁਸੀਂ ਯੋਗ ਹੋਵੋਗੇ:

### ਤਕਨੀਕੀ ਮਾਹਰਤਾ
- **ਤੈਨਾਤ ਅਤੇ ਪ੍ਰਬੰਧਨ**: ਵਿਕਾਸ ਅਤੇ ਪ੍ਰੋਡਕਸ਼ਨ ਮਾਹੌਲਾਂ ਵਿੱਚ Foundry Local ਸਥਾਪਨਾਵਾਂ
- **ਮਾਡਲ ਇੰਟੀਗ੍ਰੇਸ਼ਨ**: ਮਾਈਕਰੋਸਾਫਟ, Hugging Face, ਅਤੇ ਕਮਿਊਨਿਟੀ ਸਰੋਤਾਂ ਤੋਂ ਵਿਭਿੰਨ ਮਾਡਲ ਪਰਿਵਾਰਾਂ ਨਾਲ ਸਹਜਤਾ ਨਾਲ ਕੰਮ ਕਰੋ
- **ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਓ**: ਉੱਨਤ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਅਤੇ ਅਪਟਿਮਾਈਜ਼ੇਸ਼ਨ ਨਾਲ ਪ੍ਰੋਡਕਸ਼ਨ-ਤਿਆਰ AI ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਓ
- **ਏਜੰਟ ਵਿਕਸਿਤ ਕਰੋ**: ਗ੍ਰਾਊਂਡਿੰਗ, ਤਰਕ, ਅਤੇ ਟੂਲ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਨਾਲ ਸੁਧਾਰਤ AI ਏਜੰਟ ਲਾਗੂ ਕਰੋ

### ਰਣਨੀਤਿਕ ਸਮਝ
- **ਆਰਕੀਟੈਕਚਰ ਫੈਸਲੇ**: ਸਥਾਨਕ ਅਤੇ ਕਲਾਉਡ ਤੈਨਾਤੀ ਵਿਚਕਾਰ ਜਾਣਕਾਰੀ ਵਾਲੇ ਚੋਣਾਂ ਕਰੋ
- **ਪ੍ਰਦਰਸ਼ਨ ਅਪਟਿਮਾਈਜ਼ੇਸ਼ਨ**: ਵੱਖ-ਵੱਖ ਹਾਰਡਵੇਅਰ ਸੰਰਚਨਾਵਾਂ ਵਿੱਚ ਇੰਫਰੈਂਸ ਪ੍ਰਦਰਸ਼ਨ ਨੂੰ ਅਪਟਿਮਾਈਜ਼ ਕਰੋ
- **ਕਾਰੋਬਾਰੀ ਸਕੇਲਿੰਗ**: ਸਥਾਨਕ ਪ੍ਰੋਟੋਟਾਈਪ ਤੋਂ ਕਾਰੋਬਾਰੀ ਤੈਨਾਤੀ ਤੱਕ ਐਪਲੀਕੇਸ਼ਨ ਡਿਜ਼ਾਈਨ ਕਰੋ
- **ਗੋਪਨੀਯਤਾ ਅਤੇ ਸੁਰੱਖਿਆ**: ਸਥਾਨਕ ਇੰਫਰੈਂਸ ਨਾਲ ਗੋਪਨੀਯਤਾ-ਸੰਭਾਲ AI ਹੱਲ ਲਾਗੂ ਕਰੋ

### ਨਵੀਨਤਾ ਯੋਗਤਾਵਾਂ
- **ਤੇਜ਼ ਪ੍ਰੋਟੋਟਾਈਪਿੰਗ**: AI ਐਪਲੀਕੇਸ਼ਨ ਸੰਕਲਪਾਂ ਨੂੰ ਤੇਜ਼ੀ ਨਾਲ ਬਣਾਓ ਅਤੇ ਟੈਸਟ ਕਰੋ
- **ਕਮਿਊਨਿਟੀ ਇੰਟੀਗ੍ਰੇਸ਼ਨ**: ਖੁੱਲ੍ਹੇ-ਸਰੋਤ ਮਾਡਲਾਂ ਦਾ ਲਾਭ ਉਠਾਓ ਅਤੇ ਪਰਿਵਾਰ ਵਿੱਚ ਯੋਗਦਾਨ ਪਾਓ
- **ਉੱਨਤ ਪੈਟਰਨ**: RAG, ਏਜੰਟ, ਅਤੇ ਟੂਲ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਸਮੇਤ ਕੱਟਿੰਗ-ਐਜ AI ਪੈਟਰਨ ਲਾਗੂ ਕਰੋ
- **ਭਵਿੱਖ-ਤਿਆਰ ਵਿਕਾਸ**: ਉਭਰ ਰਹੀਆਂ AI ਤਕਨਾਲੋਜੀਆਂ ਅਤੇ ਪੈਟਰਨ ਲਈ ਤਿਆਰ ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਓ
