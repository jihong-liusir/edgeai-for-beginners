# Module 08: Microsoft Foundry Local ਨਾਲ ਹੱਥ-ਅਜਮਾਈ - ਪੂਰਾ ਡਿਵੈਲਪਰ ਟੂਲਕਿਟ

## ਝਲਕ

Microsoft Foundry Local ਐਜ AI ਵਿਕਾਸ ਦੀ ਅਗਲੀ ਪੀੜੀ ਨੂੰ ਦਰਸਾਉਂਦਾ ਹੈ, ਜੋ ਡਿਵੈਲਪਰਾਂ ਨੂੰ ਤਾਕਤਵਰ ਟੂਲ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ ਤਾਂ ਜੋ ਉਹ AI ਐਪਲੀਕੇਸ਼ਨ ਨੂੰ ਸਥਾਨਕ ਤੌਰ 'ਤੇ ਬਣਾਉਣ, ਤੈਨਾਤ ਕਰਨ ਅਤੇ ਵਧਾ ਸਕਣ। ਇਹ ਸਾਰੇ ਪ੍ਰਕਿਰਿਆ ਨੂੰ Azure AI Foundry ਨਾਲ ਬੇਹਤਰੀਨ ਤਰੀਕੇ ਨਾਲ ਜੋੜਦਾ ਹੈ। ਇਸ ਮਾਡਿਊਲ ਵਿੱਚ Foundry Local ਦੀ ਪੂਰੀ ਜਾਣਕਾਰੀ ਦਿੱਤੀ ਗਈ ਹੈ, ਇੰਸਟਾਲੇਸ਼ਨ ਤੋਂ ਲੈ ਕੇ ਉੱਚ-ਸਤਹ ਦੇ ਏਜੰਟ ਵਿਕਾਸ ਤੱਕ।

**ਮੁੱਖ ਤਕਨਾਲੋਜੀਆਂ:**
- Microsoft Foundry Local CLI ਅਤੇ SDK
- Azure AI Foundry ਇੰਟੀਗ੍ਰੇਸ਼ਨ
- ਡਿਵਾਈਸ 'ਤੇ ਮਾਡਲ ਇੰਫਰੈਂਸ
- ਸਥਾਨਕ ਮਾਡਲ ਕੈਸ਼ਿੰਗ ਅਤੇ ਅਪਟਿਮਾਈਜ਼ੇਸ਼ਨ
- ਏਜੰਟ-ਅਧਾਰਿਤ ਆਰਕੀਟੈਕਚਰ

## ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

ਇਸ ਮਾਡਿਊਲ ਨੂੰ ਪੂਰਾ ਕਰਨ ਨਾਲ, ਤੁਸੀਂ:

- **Foundry Local ਵਿੱਚ ਮਾਹਰ ਬਣੋ**: Windows 11 ਵਿਕਾਸ ਲਈ ਇੰਸਟਾਲ, ਕਨਫਿਗਰ ਅਤੇ ਅਪਟਿਮਾਈਜ਼ ਕਰੋ
- **ਵੱਖ-ਵੱਖ ਮਾਡਲ ਤੈਨਾਤ ਕਰੋ**: CLI ਕਮਾਂਡਾਂ ਨਾਲ phi, qwen, deepseek, ਅਤੇ GPT ਮਾਡਲ ਸਥਾਨਕ ਤੌਰ 'ਤੇ ਚਲਾਓ
- **ਪ੍ਰੋਡਕਸ਼ਨ ਹੱਲ ਬਣਾਓ**: ਉੱਚ-ਸਤਹ ਦੇ ਪ੍ਰੋਮਪਟ ਇੰਜੀਨੀਅਰਿੰਗ ਅਤੇ ਡਾਟਾ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਨਾਲ AI ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਓ
- **ਖੁੱਲ੍ਹੇ-ਸਰੋਤ ਪਲੇਟਫਾਰਮ ਦਾ ਲਾਭ ਲਵੋ**: Hugging Face ਮਾਡਲ ਅਤੇ ਕਮਿਊਨਿਟੀ ਯੋਗਦਾਨਾਂ ਨੂੰ ਜੋੜੋ
- **AI ਏਜੰਟ ਵਿਕਸਿਤ ਕਰੋ**: ਗ੍ਰਾਊਂਡਿੰਗ ਅਤੇ ਆਰਕਸਟ੍ਰੇਸ਼ਨ ਸਮਰਥਾ ਵਾਲੇ ਬੁੱਧੀਮਾਨ ਏਜੰਟ ਬਣਾਓ
- **ਇੰਟਰਪ੍ਰਾਈਜ਼ ਪੈਟਰਨ ਲਾਗੂ ਕਰੋ**: ਪ੍ਰੋਡਕਸ਼ਨ ਤੈਨਾਤੀ ਲਈ ਮਾਡੂਲਰ, ਸਕੇਲ ਕਰਨ ਯੋਗ AI ਹੱਲ ਬਣਾਓ

## ਪੂਰਵ ਸ਼ਰਤਾਂ

### ਸਿਸਟਮ ਦੀਆਂ ਲੋੜਾਂ
- **ਓਪਰੇਟਿੰਗ ਸਿਸਟਮ**: Windows 11 (22H2 ਜਾਂ ਇਸ ਤੋਂ ਬਾਅਦ)
- **ਮੈਮਰੀ**: 16GB RAM (ਵੱਡੇ ਮਾਡਲਾਂ ਲਈ 32GB ਦੀ ਸਿਫਾਰਸ਼ ਕੀਤੀ ਗਈ ਹੈ)
- **ਸਟੋਰੇਜ**: ਮਾਡਲ ਕੈਸ਼ਿੰਗ ਲਈ 50GB ਖਾਲੀ ਜਗ੍ਹਾ
- **ਹਾਰਡਵੇਅਰ**: NPU-ਸਮਰਥਿਤ ਡਿਵਾਈਸ ਦੀ ਸਿਫਾਰਸ਼ ਕੀਤੀ ਗਈ (Copilot+ PC), GPU ਵਿਕਲਪਿਕ
- **ਨੈਟਵਰਕ**: ਸ਼ੁਰੂਆਤੀ ਮਾਡਲ ਡਾਊਨਲੋਡ ਲਈ ਉੱਚ-ਗਤੀ ਇੰਟਰਨੈਟ

### ਵਿਕਾਸ ਦਾ ਮਾਹੌਲ
- Visual Studio Code AI Toolkit ਐਕਸਟੈਂਸ਼ਨ ਨਾਲ
- Python 3.10+ ਅਤੇ pip
- Git ਵਰਜਨ ਕੰਟਰੋਲ ਲਈ
- PowerShell ਜਾਂ Command Prompt
- Azure CLI (ਕਲਾਉਡ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਲਈ ਵਿਕਲਪਿਕ)

### ਗਿਆਨ ਦੀਆਂ ਪੂਰਵ ਸ਼ਰਤਾਂ
- AI/ML ਸੰਕਲਪਾਂ ਦੀ ਬੁਨਿਆਦੀ ਸਮਝ
- ਕਮਾਂਡ ਲਾਈਨ ਨਾਲ ਜਾਣੂ
- Python ਪ੍ਰੋਗਰਾਮਿੰਗ ਬੁਨਿਆਦੀਆਂ
- REST API ਸੰਕਲਪ
- ਪ੍ਰੋਮਪਟਿੰਗ ਅਤੇ ਮਾਡਲ ਇੰਫਰੈਂਸ ਦੀ ਬੁਨਿਆਦੀ ਜਾਣਕਾਰੀ

## ਸਿੱਖਣ ਦੇ ਨਤੀਜੇ

ਇਸ ਮਾਡਿਊਲ ਨੂੰ ਪੂਰਾ ਕਰਨ ਤੋਂ ਬਾਅਦ, ਤੁਸੀਂ ਯੋਗ ਹੋਵੋਗੇ:

### ਤਕਨੀਕੀ ਮਾਹਰਤਾ
- **ਤੈਨਾਤ ਅਤੇ ਪ੍ਰਬੰਧਨ**: ਵਿਕਾਸ ਅਤੇ ਪ੍ਰੋਡਕਸ਼ਨ ਮਾਹੌਲ ਵਿੱਚ Foundry Local ਇੰਸਟਾਲੇਸ਼ਨ
- **ਮਾਡਲ ਇੰਟੀਗ੍ਰੇਸ਼ਨ**: Microsoft, Hugging Face, ਅਤੇ ਕਮਿਊਨਿਟੀ ਸਰੋਤਾਂ ਦੇ ਵੱਖ-ਵੱਖ ਮਾਡਲਾਂ ਨਾਲ ਸਹਜਤਾ ਨਾਲ ਕੰਮ ਕਰੋ
- **ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਓ**: ਉੱਚ-ਗੁਣਵੱਤਾ ਵਾਲੇ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਅਤੇ ਅਪਟਿਮਾਈਜ਼ੇਸ਼ਨ ਨਾਲ ਪ੍ਰੋਡਕਸ਼ਨ-ਤਿਆਰ AI ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਓ
- **ਏਜੰਟ ਵਿਕਸਿਤ ਕਰੋ**: ਗ੍ਰਾਊਂਡਿੰਗ, ਤਰਕ, ਅਤੇ ਟੂਲ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਨਾਲ ਉੱਚ-ਸਤਹ ਦੇ AI ਏਜੰਟ ਲਾਗੂ ਕਰੋ

### ਰਣਨੀਤਿਕ ਸਮਝ
- **ਆਰਕੀਟੈਕਚਰ ਫੈਸਲੇ**: ਸਥਾਨਕ ਅਤੇ ਕਲਾਉਡ ਤੈਨਾਤੀ ਵਿਚਕਾਰ ਸੂਝਵਾਨ ਚੋਣ ਕਰੋ
- **ਪ੍ਰਦਰਸ਼ਨ ਅਪਟਿਮਾਈਜ਼ੇਸ਼ਨ**: ਵੱਖ-ਵੱਖ ਹਾਰਡਵੇਅਰ ਸੰਰਚਨਾਵਾਂ 'ਤੇ ਇੰਫਰੈਂਸ ਪ੍ਰਦਰਸ਼ਨ ਨੂੰ ਅਪਟਿਮਾਈਜ਼ ਕਰੋ
- **ਇੰਟਰਪ੍ਰਾਈਜ਼ ਸਕੇਲਿੰਗ**: ਸਥਾਨਕ ਪ੍ਰੋਟੋਟਾਈਪ ਤੋਂ ਇੰਟਰਪ੍ਰਾਈਜ਼ ਤੈਨਾਤੀ ਤੱਕ ਸਕੇਲ ਕਰਨ ਯੋਗ ਐਪਲੀਕੇਸ਼ਨ ਡਿਜ਼ਾਈਨ ਕਰੋ
- **ਗੋਪਨੀਯਤਾ ਅਤੇ ਸੁਰੱਖਿਆ**: ਸਥਾਨਕ ਇੰਫਰੈਂਸ ਨਾਲ ਗੋਪਨੀ
ਇਹ ਮੋਡਿਊਲ ਐਜ AI ਵਿਕਾਸ ਦੇ ਅਗਲੇ ਪੱਧਰ ਨੂੰ ਦਰਸਾਉਂਦਾ ਹੈ, ਜਿਸ ਵਿੱਚ Microsoft ਦੇ ਉੱਚ-ਪੱਧਰੀ ਟੂਲਾਂ ਨੂੰ ਖੁੱਲ੍ਹੇ-ਸਰੋਤ ਪਰਿਸਰ ਦੇ ਲਚਕ ਅਤੇ ਨਵੀਨਤਾ ਨਾਲ ਜੋੜਿਆ ਗਿਆ ਹੈ। Foundry Local ਦੇ ਸਾਰੇ 10 ਵਿਸਤ੍ਰਿਤ ਨਮੂਨਿਆਂ ਨੂੰ ਸਮਝ ਕੇ, ਤੁਸੀਂ AI ਐਪਲੀਕੇਸ਼ਨ ਵਿਕਾਸ ਦੇ ਅਗਲੇ ਪੱਧਰ 'ਤੇ ਪਹੁੰਚ ਜਾਵੋਗੇ।
