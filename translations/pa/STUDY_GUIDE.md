## ਪੇਸ਼ਕਸ਼

ਐਜਏਆਈ ਬਿਗਿਨਰਜ਼ ਸਟਡੀ ਗਾਈਡ ਵਿੱਚ ਤੁਹਾਡਾ ਸਵਾਗਤ ਹੈ! ਇਹ ਦਸਤਾਵੇਜ਼ ਤੁਹਾਨੂੰ ਕੋਰਸ ਸਮੱਗਰੀ ਨੂੰ ਪ੍ਰਭਾਵਸ਼ਾਲੀ ਢੰਗ ਨਾਲ ਸਮਝਣ ਅਤੇ ਸਿੱਖਣ ਦੇ ਅਨੁਭਵ ਨੂੰ ਵਧਾਉਣ ਵਿੱਚ ਮਦਦ ਕਰਨ ਲਈ ਤਿਆਰ ਕੀਤਾ ਗਿਆ ਹੈ। ਇਹ ਸਟ੍ਰਕਚਰਡ ਸਿੱਖਣ ਦੇ ਰਾਹ, ਸੁਝਾਏ ਅਧਿਐਨ ਸਮਾਂ-ਸਾਰਣੀ, ਮੁੱਖ ਧਾਰਨਾ ਸੰਖੇਪ ਅਤੇ ਐਜਏਆਈ ਤਕਨਾਲੋਜੀਆਂ ਦੀ ਸਮਝ ਨੂੰ ਗਹਿਰਾ ਕਰਨ ਲਈ ਸਹਾਇਕ ਸਰੋਤ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ।

ਇਹ ਇੱਕ ਸੰਖੇਪ 20-ਘੰਟੇ ਦਾ ਕੋਰਸ ਹੈ ਜੋ ਐਜਏਆਈ ਬਾਰੇ ਜ਼ਰੂਰੀ ਗਿਆਨ ਨੂੰ ਸਮੇਂ-ਕੁਸ਼ਲ ਫਾਰਮੈਟ ਵਿੱਚ ਪੇਸ਼ ਕਰਦਾ ਹੈ, ਜੋ ਕਿ ਵਿਅਸਤ ਪੇਸ਼ੇਵਰਾਂ ਅਤੇ ਵਿਦਿਆਰਥੀਆਂ ਲਈ ਬਹੁਤ ਹੀ ਉਚਿਤ ਹੈ ਜੋ ਇਸ ਉਭਰਦੇ ਖੇਤਰ ਵਿੱਚ ਤੁਰੰਤ ਵਿਵਹਾਰਕ ਹੁਨਰ ਪ੍ਰਾਪਤ ਕਰਨਾ ਚਾਹੁੰਦੇ ਹਨ।

## ਕੋਰਸ ਝਲਕ

ਇਹ ਕੋਰਸ ਸੱਤ ਵਿਸਤ੍ਰਿਤ ਮੋਡਿਊਲਾਂ ਵਿੱਚ ਵਿਭਾਜਿਤ ਹੈ:

1. **ਐਜਏਆਈ ਮੂਲ ਭੂਤ ਅਤੇ ਰੂਪਾਂਤਰਣ** - ਮੁੱਖ ਧਾਰਨਾਵਾਂ ਅਤੇ ਤਕਨਾਲੋਜੀ ਬਦਲਾਅ ਨੂੰ ਸਮਝਣਾ  
2. **ਛੋਟੇ ਭਾਸ਼ਾ ਮਾਡਲ ਬੁਨਿਆਦਾਂ** - ਵੱਖ-ਵੱਖ SLM ਪਰਿਵਾਰਾਂ ਅਤੇ ਉਨ੍ਹਾਂ ਦੇ ਆਰਕੀਟੈਕਚਰ ਦੀ ਖੋਜ  
3. **ਛੋਟੇ ਭਾਸ਼ਾ ਮਾਡਲ ਡਿਪਲੌਇਮੈਂਟ** - ਵਿਵਹਾਰਕ ਡਿਪਲੌਇਮੈਂਟ ਰਣਨੀਤੀਆਂ ਨੂੰ ਲਾਗੂ ਕਰਨਾ  
4. **ਮਾਡਲ ਫਾਰਮੈਟ ਕਨਵਰਜ਼ਨ ਅਤੇ ਕੁਆੰਟਾਈਜ਼ੇਸ਼ਨ** - 6 ਫਰੇਮਵਰਕਾਂ ਨਾਲ ਉੱਚ-ਸਤਰੀਅ ਅਪਟਿਮਾਈਜ਼ੇਸ਼ਨ  
5. **SLMOps - ਛੋਟੇ ਭਾਸ਼ਾ ਮਾਡਲ ਓਪਰੇਸ਼ਨ** - ਉਤਪਾਦਨ ਜੀਵਨਚੱਕਰ ਪ੍ਰਬੰਧਨ ਅਤੇ ਡਿਪਲੌਇਮੈਂਟ  
6. **SLM ਏਜੰਟਿਕ ਸਿਸਟਮ** - AI ਏਜੰਟ, ਫੰਕਸ਼ਨ ਕਾਲਿੰਗ ਅਤੇ ਮਾਡਲ ਸੰਦਰਭ ਪ੍ਰੋਟੋਕੋਲ  
7. **ਐਜਏਆਈ ਇੰਪਲੀਮੈਂਟੇਸ਼ਨ ਨਮੂਨੇ** - AI ਟੂਲਕਿਟ, Windows ਵਿਕਾਸ ਅਤੇ ਪਲੇਟਫਾਰਮ-ਵਿਸ਼ੇਸ਼ ਇੰਪਲੀਮੈਂਟੇਸ਼ਨ  

## ਇਸ ਸਟਡੀ ਗਾਈਡ ਨੂੰ ਕਿਵੇਂ ਵਰਤਣਾ ਹੈ

- **ਤਰੱਕੀਵਾਦੀ ਸਿੱਖਣ**: ਸਭ ਤੋਂ ਸੰਗਠਿਤ ਸਿੱਖਣ ਦੇ ਅਨੁਭਵ ਲਈ ਮੋਡਿਊਲਾਂ ਨੂੰ ਕ੍ਰਮਵਾਰ ਅਨੁਸਰਣ ਕਰੋ  
- **ਗਿਆਨ ਚੈੱਕਪੋਇੰਟ**: ਹਰ ਸੈਕਸ਼ਨ ਦੇ ਬਾਅਦ ਸਵੈ-ਮੁਲਾਂਕਣ ਪ੍ਰਸ਼ਨਾਂ ਦੀ ਵਰਤੋਂ ਕਰੋ  
- **ਹੈਂਡਸ-ਆਨ ਅਭਿਆਸ**: ਸਿਧਾਂਤਕ ਧਾਰਨਾਵਾਂ ਨੂੰ ਮਜ਼ਬੂਤ ਕਰਨ ਲਈ ਸੁਝਾਏ ਅਭਿਆਸ ਪੂਰੇ ਕਰੋ  
- **ਸਹਾਇਕ ਸਰੋਤ**: ਉਹ ਵਿਸ਼ੇ ਜੋ ਤੁਹਾਨੂੰ ਸਭ ਤੋਂ ਵੱਧ ਦਿਲਚਸਪ ਲੱਗਦੇ ਹਨ, ਉਨ੍ਹਾਂ ਲਈ ਵਾਧੂ ਸਮੱਗਰੀ ਦੀ ਖੋਜ ਕਰੋ

## Module 6: SLM Agentic Systems - AI Agents ਅਤੇ Function Calling

### ਮੁੱਖ ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

- ਛੋਟੇ ਭਾਸ਼ਾ ਮਾਡਲਾਂ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਐਜ ਵਾਤਾਵਰਣਾਂ ਲਈ ਬੁੱਧੀਮਾਨ AI ਏਜੰਟ ਬਣਾਓ

- ਮਾਡਲ ਕਾਂਟੈਕਸਟ ਪ੍ਰੋਟੋਕੋਲ (MCP) ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਵਿੱਚ ਮਾਹਰ ਬਣੋ
- ਘੱਟ ਤੋਂ ਘੱਟ ਮਨੁੱਖੀ ਦਖਲ ਨਾਲ ਉੱਨਤ agentic systems ਬਣਾਓ

### ਅਧਿਐਨ ਫੋਕਸ ਖੇਤਰ

#### ਸੈਕਸ਼ਨ 1: AI Agents ਅਤੇ SLM Foundations
- **ਪ੍ਰਾਥਮਿਕਤਾ ਸੰਕਲਪ**: 
  - ਏਜੰਟ ਵਰਗੀਕਰਨ ਫਰੇਮਵਰਕ (reflex, model-based, goal-based, learning agents)
  - SLM ਅਤੇ LLM ਦੇ trade-offs ਦਾ ਵਿਸ਼ਲੇਸ਼ਣ
  - ਐਜ-ਖਾਸ agent design patterns
  - ਏਜੰਟਾਂ ਲਈ ਸਰੋਤ ਅਪਟਿਮਾਈਜ਼ੇਸ਼ਨ

#### ਸੈਕਸ਼ਨ 2: Function Calling in Small Language Models
- **ਪ੍ਰਾਥਮਿਕਤਾ ਸੰਕਲਪ**: 
  - systematic workflow ਲਾਗੂਅਤ (intent detection, JSON output, external execution)
  - ਪਲੇਟਫਾਰਮ-ਖਾਸ implementations (Phi-4-mini, selected Qwen models, Microsoft Foundry Local)
  - ਉੱਨਤ ਉਦਾਹਰਨ (multi-agent collaboration, dynamic tool selection)
  - ਉਤਪਾਦਨ ਵਿਚਾਰ (rate limiting, audit logging, security measures)

  - ਪ੍ਰੋਟੋਕੋਲ ਆਰਕੀਟੈਕਚਰ ਅਤੇ layered system design
  - multi-backend support (Ollama for development, vLLM for production)
  - connection protocols (STDIO ਅਤੇ SSE modes)
  - ਹਕੀਕਤੀ-world applications (web automation, data processing, API integration)

3. Agent communication ਵਿੱਚ Model Context Protocol ਦੀ ਭੂਮਿਕਾ ਸਮਝਾਓ।

### ਹੱਥ-ਅਜਮਾਇਸ਼ ਅਭਿਆਸ
