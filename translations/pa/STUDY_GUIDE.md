## ਪਰਿਚਯ

ਐਜਏਆਈ ਸ਼ੁਰੂਆਤੀਆਂ ਲਈ ਸਟੱਡੀ ਗਾਈਡ ਵਿੱਚ ਤੁਹਾਡਾ ਸਵਾਗਤ ਹੈ! ਇਹ ਦਸਤਾਵੇਜ਼ ਤੁਹਾਨੂੰ ਕੋਰਸ ਸਮੱਗਰੀ ਨੂੰ ਪ੍ਰਭਾਵਸ਼ਾਲੀ ਢੰਗ ਨਾਲ ਸਮਝਣ ਅਤੇ ਆਪਣੇ ਸਿੱਖਣ ਦੇ ਅਨੁਭਵ ਨੂੰ ਵਧਾਉਣ ਵਿੱਚ ਮਦਦ ਕਰਨ ਲਈ ਤਿਆਰ ਕੀਤਾ ਗਿਆ ਹੈ। ਇਸ ਵਿੱਚ ਢਾਂਚੇਬੱਧ ਸਿੱਖਣ ਦੇ ਰਸਤੇ, ਸੁਝਾਏ ਗਏ ਅਧਿਐਨ ਸ਼ਡਿਊਲ, ਮੁੱਖ ਸਿਧਾਂਤਾਂ ਦੇ ਸਾਰ, ਅਤੇ ਐਜਏਆਈ ਤਕਨਾਲੋਜੀਆਂ ਦੀ ਗਹਿਰਾਈ ਨਾਲ ਸਮਝਣ ਲਈ ਪੂਰਕ ਸਾਧਨ ਸ਼ਾਮਲ ਹਨ।

ਇਹ ਇੱਕ ਸੰਖੇਪ 20-ਘੰਟੇ ਦਾ ਕੋਰਸ ਹੈ ਜੋ ਐਜਏਆਈ ਬਾਰੇ ਜ਼ਰੂਰੀ ਗਿਆਨ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ, ਜੋ ਵਿਅਸਤ ਪੇਸ਼ੇਵਰਾਂ ਅਤੇ ਵਿਦਿਆਰਥੀਆਂ ਲਈ ਬਹੁਤ ਹੀ ਉਚਿਤ ਹੈ ਜੋ ਇਸ ਉਭਰ ਰਹੀ ਖੇਤਰ ਵਿੱਚ ਜਲਦੀ ਪ੍ਰਯੋਗਿਕ ਹੁਨਰ ਪ੍ਰਾਪਤ ਕਰਨਾ ਚਾਹੁੰਦੇ ਹਨ।

## ਕੋਰਸ ਝਲਕ

ਇਹ ਕੋਰਸ ਅੱਠ ਵਿਸਤ੍ਰਿਤ ਮਾਡਿਊਲਾਂ ਵਿੱਚ ਵਿਭਾਜਿਤ ਹੈ:

0. **ਐਜਏਆਈ ਦਾ ਪਰਿਚਯ** - ਉਦਯੋਗੀ ਐਪਲੀਕੇਸ਼ਨਾਂ ਅਤੇ ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼ਾਂ ਨਾਲ ਬੁਨਿਆਦ ਅਤੇ ਸੰਦਰਭ ਸੈਟਿੰਗ
1. **ਐਜਏਆਈ ਦੇ ਮੁੱਢਲੇ ਸਿਧਾਂਤ ਅਤੇ ਬਦਲਾਅ** - ਮੁੱਖ ਸਿਧਾਂਤਾਂ ਅਤੇ ਤਕਨਾਲੋਜੀ ਬਦਲਾਅ ਨੂੰ ਸਮਝਣਾ
2. **ਛੋਟੇ ਭਾਸ਼ਾ ਮਾਡਲ ਦੀ ਬੁਨਿਆਦ** - ਵੱਖ-ਵੱਖ SLM ਪਰਿਵਾਰਾਂ ਅਤੇ ਉਨ੍ਹਾਂ ਦੀਆਂ ਆਰਕੀਟੈਕਚਰਾਂ ਦੀ ਖੋਜ
3. **ਛੋਟੇ ਭਾਸ਼ਾ ਮਾਡਲ ਡਿਪਲੌਇਮੈਂਟ** - ਪ੍ਰਯੋਗਿਕ ਡਿਪਲੌਇਮੈਂਟ ਰਣਨੀਤੀਆਂ ਨੂੰ ਲਾਗੂ ਕਰਨਾ
4. **ਮਾਡਲ ਫਾਰਮੈਟ ਕਨਵਰਜ਼ਨ ਅਤੇ ਕੁਆੰਟਾਈਜ਼ੇਸ਼ਨ** - 6 ਫਰੇਮਵਰਕਾਂ ਨਾਲ ਉੱਚ-ਪੱਧਰੀ ਅਪਟੀਮਾਈਜ਼ੇਸ਼ਨ
5. **SLMOps - ਛੋਟੇ ਭਾਸ਼ਾ ਮਾਡਲ ਦੇ ਸੰਚਾਲਨ** - ਉਤਪਾਦਨ ਜੀਵਨਚੱਕਰ ਪ੍ਰਬੰਧਨ ਅਤੇ ਡਿਪਲੌਇਮੈਂਟ
6. **SLM ਏਜੰਟਿਕ ਸਿਸਟਮ** - AI ਏਜੰਟ, ਫੰਕਸ਼ਨ ਕਾਲਿੰਗ, ਅਤੇ ਮਾਡਲ ਸੰਦਰਭ ਪ੍ਰੋਟੋਕੋਲ
7. **ਐਜਏਆਈ ਇੰਪਲੀਮੈਂਟੇਸ਼ਨ ਸੈਂਪਲ** - AI ਟੂਲਕਿਟ, Windows ਡਿਵੈਲਪਮੈਂਟ, ਅਤੇ ਪਲੇਟਫਾਰਮ-ਵਿਸ਼ੇਸ਼ ਇੰਪਲੀਮੈਂਟੇਸ਼ਨ
8. **ਮਾਈਕਰੋਸਾਫਟ ਫਾਊਂਡਰੀ ਲੋਕਲ – ਪੂਰਾ ਡਿਵੈਲਪਰ ਟੂਲਕਿਟ** - ਹਾਈਬ੍ਰਿਡ ਐਜ਼ਰ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਨਾਲ ਲੋਕਲ-ਪਹਿਲਾ ਵਿਕਾਸ (ਮਾਡਿਊਲ 08)

## ਇਸ ਸਟੱਡੀ ਗਾਈਡ ਨੂੰ ਕਿਵੇਂ ਵਰਤਣਾ ਹੈ

- **ਤਰੱਕੀਵਾਦੀ ਸਿੱਖਣ**: ਸਭ ਤੋਂ ਸੰਗਠਿਤ ਸਿੱਖਣ ਦੇ ਅਨੁਭਵ ਲਈ ਮਾਡਿਊਲਾਂ ਨੂੰ ਕ੍ਰਮਵਾਰ ਪੂਰਾ ਕਰੋ
- **ਗਿਆਨ ਦੇ ਚੌਕੀਦਾਰ**: ਹਰ ਭਾਗ ਦੇ ਬਾਅਦ ਸਵੈ-ਮੁਲਾਂਕਣ ਪ੍ਰਸ਼ਨਾਂ ਦੀ ਵਰਤੋਂ ਕਰੋ
- **ਹੈਂਡਸ-ਆਨ ਅਭਿਆਸ**: ਸਿਧਾਂਤਕ ਸਿਧਾਂਤਾਂ ਨੂੰ ਮਜ਼ਬੂਤ ਕਰਨ ਲਈ ਸੁਝਾਏ ਗਏ ਅਭਿਆਸ ਪੂਰੇ ਕਰੋ
- **ਪੂਰਕ ਸਾਧਨ**: ਉਹ ਵਿਸ਼ੇ ਜੋ ਤੁਹਾਨੂੰ ਸਭ ਤੋਂ ਵੱਧ ਦਿਲਚਸਪ ਲੱਗਦੇ ਹਨ, ਉਨ੍ਹਾਂ ਲਈ ਵਾਧੂ ਸਮੱਗਰੀ ਦੀ ਖੋਜ ਕਰੋ

### ਮੁੱਖ ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

- ਸਮਝੋ ਕਿ ਐਜਏਆਈ ਕੀ ਹੈ ਅਤੇ ਇਹ ਅੱਜ ਦੀ ਤਕਨਾਲੋਜੀ ਵਿੱਚ ਕਿਉਂ ਮਹੱਤਵਪੂਰਨ ਹੈ
- ਉਹ ਮੁੱਖ ਉਦਯੋਗ ਪਛਾਣੋ ਜਿਨ੍ਹਾਂ ਨੂੰ ਐਜਏਆਈ ਨੇ ਬਦਲ ਦਿੱਤਾ ਹੈ ਅਤੇ ਉਨ੍ਹਾਂ ਦੇ ਖਾਸ ਐਪਲੀਕੇਸ਼ਨ

  - ਲਾਗਤ ਕੁਸ਼ਲਤਾ ਅਤੇ ਪ੍ਰਾਈਵੇਸੀ-ਪਹਿਲਾਂ ਆਰਕੀਟੈਕਚਰ  
  - ਰਣਨੀਤਿਕ ਕਾਰੋਬਾਰੀ ਪ੍ਰਭਾਵ ਅਤੇ ਮੁਕਾਬਲੇ ਦੇ ਫਾਇਦੇ

## ਮਾਡਿਊਲ 6: SLM Agentic Systems - AI Agents ਅਤੇ Function Calling

### ਮੁੱਖ ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

- ਮਾਡਲ ਕਾਂਟੈਕਸਟ ਪ੍ਰੋਟੋਕੋਲ (MCP) ਇੰਟਿਗ੍ਰੇਸ਼ਨ ਵਿੱਚ ਮਾਹਰ ਬਣੋ  
- ਘੱਟ ਮਨੁੱਖੀ ਦਖਲ ਨਾਲ ਸੁਧਾਰਤ ਏਜੰਟਿਕ ਸਿਸਟਮ ਬਣਾਓ  

### ਅਧਿਐਨ ਫੋਕਸ ਖੇਤਰ

#### ਸੈਕਸ਼ਨ 1: AI Agents ਅਤੇ SLM ਦੀਆਂ ਬੁਨਿਆਦਾਂ  
- **ਪ੍ਰਾਇਰਟੀ ਧਾਰਨਾਵਾਂ**:  
  - ਏਜੰਟ ਵਰਗੀਕਰਨ ਫਰੇਮਵਰਕ (ਰਿਫਲੈਕਸ, ਮਾਡਲ-ਅਧਾਰਿਤ, ਗੋਲ-ਅਧਾਰਿਤ, ਸਿੱਖਣ ਵਾਲੇ ਏਜੰਟ)  
  - SLM ਅਤੇ LLM ਦੀ ਤੁਲਨਾ  
  - Edge-ਵਿਸ਼ੇਸ਼ ਏਜੰਟ ਡਿਜ਼ਾਈਨ ਪੈਟਰਨ  
  - ਏਜੰਟਾਂ ਲਈ ਸਰੋਤ ਅਪਟਿਮਾਈਜ਼ੇਸ਼ਨ  

#### ਸੈਕਸ਼ਨ 2: Function Calling in Small Language Models  
- **ਪ੍ਰਾਇਰਟੀ ਧਾਰਨਾਵਾਂ**:  
  - systematic workflow ਲਾਗੂ ਕਰਨਾ (intent detection, JSON output, external execution)  
  - ਪਲੇਟਫਾਰਮ-ਵਿਸ਼ੇਸ਼ ਲਾਗੂ ਕਰਨਾ (Phi-4-mini, ਚੁਣੇ ਹੋਏ Qwen ਮਾਡਲ, Microsoft Foundry Local)  
  - ਉੱਚ-ਸਤਹ ਉਦਾਹਰਨ (multi-agent collaboration, dynamic tool selection)  
  - ਪ੍ਰੋਡਕਸ਼ਨ ਵਿਚਾਰ (rate limiting, audit logging, security measures)  

#### ਸੈਕਸ਼ਨ 3: Model Context Protocol (MCP) Integration  
- **ਪ੍ਰਾਇਰਟੀ ਧਾਰਨਾਵਾਂ**:  
  - ਪ੍ਰੋਟੋਕੋਲ ਆਰਕੀਟੈਕਚਰ ਅਤੇ ਲੇਅਰਡ ਸਿਸਟਮ ਡਿਜ਼ਾਈਨ  
  - ਮਲਟੀ-ਬੈਕਐਂਡ ਸਹਾਇਤਾ (Ollama ਵਿਕਾਸ ਲਈ, vLLM ਪ੍ਰੋਡਕਸ਼ਨ ਲਈ)  
  - ਕਨੈਕਸ਼ਨ ਪ੍ਰੋਟੋਕੋਲ (STDIO ਅਤੇ SSE ਮੋਡ)  
  - ਅਸਲ-ਦੁਨੀਆ ਦੇ ਐਪਲੀਕੇਸ਼ਨ (web automation, data processing, API integration)  

### ਸਵੈ-ਮੁਲਾਂਕਣ ਪ੍ਰਸ਼ਨ

## ਮਿਨੀ ਪ੍ਰੋਜੈਕਟ ਵਿਚਾਰ

EdgeAI ਸੰਕਲਪਾਂ ਦਾ ਅਭਿਆਸ ਕਰਨ ਲਈ ਇਨ੍ਹਾਂ ਪ੍ਰੋਜੈਕਟਾਂ ਵਿੱਚੋਂ ਇੱਕ ਪੂਰਾ ਕਰਨ 'ਤੇ ਵਿਚਾਰ ਕਰੋ (ਹਰ ਇੱਕ 2-4 ਘੰਟਿਆਂ ਲਈ ਡਿਜ਼ਾਈਨ ਕੀਤਾ ਗਿਆ ਹੈ):

### ਸ਼ੁਰੂਆਤੀ ਪ੍ਰੋਜੈਕਟ (2-3 ਘੰਟੇ ਹਰ ਇੱਕ)
1. **Edge Text Assistant**: ਇੱਕ ਛੋਟੇ ਭਾਸ਼ਾ ਮਾਡਲ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਇੱਕ ਸਧਾਰਨ ਆਫਲਾਈਨ ਟੈਕਸਟ ਪੂਰਨ ਟੂਲ ਬਣਾਓ
2. **Model Comparison Dashboard**: ਵੱਖ-ਵੱਖ SLMs ਦੇ ਪ੍ਰਦਰਸ਼ਨ ਮਾਪਦੰਡਾਂ ਦੀ ਵਿਜ਼ੁਅਲਾਈਜ਼ੇਸ਼ਨ ਬਣਾਓ
3. **Optimization Experiment**: ਇੱਕ ਹੀ ਬੇਸ ਮਾਡਲ 'ਤੇ ਵੱਖ-ਵੱਖ ਕੁਆਂਟੀਜ਼ੇਸ਼ਨ ਪੱਧਰਾਂ ਦੇ ਪ੍ਰਭਾਵ ਨੂੰ ਮਾਪੋ

### ਮੱਧਮ ਪੱਧਰ ਦੇ ਪ੍ਰੋਜੈਕਟ (3-4 ਘੰਟੇ ਹਰ ਇੱਕ)
4. **AI Toolkit Workflow**: VS Code AI Toolkit ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਇੱਕ ਮਾਡਲ ਨੂੰ ਸ਼ੁਰੂ ਤੋਂ ਅੰਤ ਤੱਕ ਅਪਟਿਮਾਈਜ਼ ਅਤੇ ਤੈਨਾਤ ਕਰੋ
5. **Windows AI Foundry Application**: Phi Silica API ਅਤੇ NPU ਅਪਟਿਮਾਈਜ਼ੇਸ਼ਨ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਇੱਕ Windows ਐਪ ਬਣਾਓ
6. **Cross-Platform Deployment**: ਇੱਕ ਹੀ ਅਪਟਿਮਾਈਜ਼ਡ ਮਾਡਲ ਨੂੰ Windows (OpenVINO) ਅਤੇ ਮੋਬਾਈਲ (.NET MAUI) 'ਤੇ ਤੈਨਾਤ ਕਰੋ
7. **Function Calling Agent**: Edge ਦ੍ਰਿਸ਼ਾਂ ਲਈ ਫੰਕਸ਼ਨ ਕਾਲਿੰਗ ਸਮਰੱਥਾ ਨਾਲ ਇੱਕ AI ਏਜੰਟ ਬਣਾਓ

### ਉੱਚ ਪੱਧਰ ਦੇ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਪ੍ਰੋਜੈਕਟ (4-5 ਘੰਟੇ ਹਰ ਇੱਕ)
8. **OpenVINO Optimization Pipeline**: NNCF ਅਤੇ GenAI ਟੂਲਕਿਟ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਪੂਰੀ ਮਾਡਲ ਅਪਟਿਮਾਈਜ਼ੇਸ਼ਨ ਲਾਗੂ ਕਰੋ
9. **SLMOps Pipeline**: ਟ੍ਰੇਨਿੰਗ ਤੋਂ ਲੈ ਕੇ Edge ਤੈਨਾਤੀ ਤੱਕ ਪੂਰੀ ਮਾਡਲ ਲਾਈਫਸਾਈਕਲ ਲਾਗੂ ਕਰੋ
10. **Multi-Model Edge System**: Edge ਹਾਰਡਵੇਅਰ 'ਤੇ ਕੰਮ ਕਰਨ ਵਾਲੇ ਕਈ ਵਿਸ਼ੇਸ਼ ਮਾਡਲ ਤੈਨਾਤ ਕਰੋ
11. **MCP Integration System**: ਟੂਲ ਇੰਟਰੈਕਸ਼ਨ ਲਈ Model Context Protocol ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਇੱਕ ਏਜੰਟਿਕ ਸਿਸਟਮ ਬਣਾਓ
