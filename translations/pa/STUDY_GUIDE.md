## ਪੇਸ਼ਕਸ਼

EdgeAI ਸ਼ੁਰੂਆਤੀ ਲਈ ਸਵਾਗਤ ਹੈ! ਇਹ ਦਸਤਾਵੇਜ਼ ਤੁਹਾਨੂੰ ਕੋਰਸ ਸਮੱਗਰੀ ਨੂੰ ਪ੍ਰਭਾਵਸ਼ਾਲੀ ਢੰਗ ਨਾਲ ਸਮਝਣ ਅਤੇ ਆਪਣੇ ਸਿੱਖਣ ਦੇ ਅਨੁਭਵ ਨੂੰ ਵਧਾਉਣ ਵਿੱਚ ਮਦਦ ਕਰਨ ਲਈ ਤਿਆਰ ਕੀਤਾ ਗਿਆ ਹੈ। ਇਹ ਸੰਗਠਿਤ ਸਿੱਖਣ ਦੇ ਰਾਹ, ਸੁਝਾਏ ਅਧਿਐਨ ਸਮਾਂ-ਸਾਰਣੀ, ਮੁੱਖ ਸੰਕਲਪ ਸਾਰਾਂ, ਅਤੇ EdgeAI ਤਕਨਾਲੋਜੀਆਂ ਦੀ ਸਮਝ ਨੂੰ ਗਹਿਰਾ ਕਰਨ ਲਈ ਸਹਾਇਕ ਸਰੋਤ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ।

ਇਹ ਇੱਕ ਸੰਖੇਪ 20-ਘੰਟੇ ਦਾ ਕੋਰਸ ਹੈ ਜੋ EdgeAI ਬਾਰੇ ਜ਼ਰੂਰੀ ਗਿਆਨ ਨੂੰ ਸਮੇਂ-ਕੁਸ਼ਲ ਫਾਰਮੈਟ ਵਿੱਚ ਪੇਸ਼ ਕਰਦਾ ਹੈ, ਜੋ ਕਿ ਵਿਅਸਤ ਪੇਸ਼ੇਵਰਾਂ ਅਤੇ ਵਿਦਿਆਰਥੀਆਂ ਲਈ ਬਹੁਤ ਹੀ ਉਚਿਤ ਹੈ ਜੋ ਇਸ ਉਭਰ ਰਹੀ ਖੇਤਰ ਵਿੱਚ ਤੁਰੰਤ ਵਿਵਹਾਰਕ ਹੁਨਰ ਪ੍ਰਾਪਤ ਕਰਨਾ ਚਾਹੁੰਦੇ ਹਨ।

## ਕੋਰਸ ਝਲਕ

ਇਹ ਕੋਰਸ ਸੱਤ ਵਿਸਤ੍ਰਿਤ ਮੋਡਿਊਲਾਂ ਵਿੱਚ ਸੰਗਠਿਤ ਹੈ:

1. **EdgeAI ਮੂਲ ਭੂਮਿਕਾ ਅਤੇ ਰੂਪਾਂਤਰਣ** - ਮੁੱਖ ਸੰਕਲਪ ਅਤੇ ਤਕਨਾਲੋਜੀ ਬਦਲਾਅ ਨੂੰ ਸਮਝਣਾ
2. **ਛੋਟੇ ਭਾਸ਼ਾ ਮਾਡਲ Foundations** - ਵੱਖ-ਵੱਖ SLM ਪਰਿਵਾਰਾਂ ਅਤੇ ਉਨ੍ਹਾਂ ਦੇ ਆਰਕੀਟੈਕਚਰ ਦੀ ਖੋਜ
3. **ਛੋਟੇ ਭਾਸ਼ਾ ਮਾਡਲ Deployment** - ਵਿਵਹਾਰਕ Deployment ਰਣਨੀਤੀਆਂ ਨੂੰ ਲਾਗੂ ਕਰਨਾ
4. **ਮਾਡਲ ਫਾਰਮੈਟ ਰੂਪਾਂਤਰਣ ਅਤੇ Quantization** - 6 ਫਰੇਮਵਰਕਾਂ ਨਾਲ ਉੱਚਤਮ ਅਪਟਿਮਾਈਜ਼ੇਸ਼ਨ
5. **SLMOps - ਛੋਟੇ ਭਾਸ਼ਾ ਮਾਡਲ Operations** - ਉਤਪਾਦਨ ਜੀਵਨਚੱਕਰ ਪ੍ਰਬੰਧਨ ਅਤੇ Deployment
6. **SLM Agentic Systems** - AI Agents, ਫੰਕਸ਼ਨ ਕਾਲਿੰਗ, ਅਤੇ Model Context Protocol
7. **EdgeAI Implementation Samples** - AI Toolkit, Windows ਵਿਕਾਸ, ਅਤੇ ਪਲੇਟਫਾਰਮ-ਵਿਸ਼ੇਸ਼ Implementation
8. **Microsoft Foundry Local – Complete Developer Toolkit** - Local-ਪਹਿਲਾਂ ਵਿਕਾਸ ਹਾਈਬ੍ਰਿਡ Azure Integration ਨਾਲ (Module 08)

## ਮਿਨੀ ਪ੍ਰੋਜੈਕਟ ਆਈਡੀਆ

EdgeAI ਸੰਕਲਪਾਂ ਦਾ ਅਭਿਆਸ ਕਰਨ ਲਈ ਹੇਠਾਂ ਦਿੱਤੇ ਪ੍ਰੋਜੈਕਟਾਂ ਵਿੱਚੋਂ ਇੱਕ ਪੂਰਾ ਕਰਨ ਬਾਰੇ ਸੋਚੋ (ਹਰ ਇੱਕ 2-4 ਘੰਟਿਆਂ ਲਈ ਡਿਜ਼ਾਈਨ ਕੀਤਾ ਗਿਆ ਹੈ):

### ਸ਼ੁਰੂਆਤੀ ਪ੍ਰੋਜੈਕਟ (ਹਰ ਇੱਕ 2-3 ਘੰਟੇ)
1. **Edge ਟੈਕਸਟ ਅਸਿਸਟੈਂਟ**: ਇੱਕ ਛੋਟੇ ਭਾਸ਼ਾ ਮਾਡਲ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਸਧਾਰਨ ਆਫਲਾਈਨ ਟੈਕਸਟ ਪੂਰਨ ਟੂਲ ਬਣਾਓ
2. **ਮਾਡਲ ਤੁਲਨਾ ਡੈਸ਼ਬੋਰਡ**: ਵੱਖ-ਵੱਖ SLMs ਦੇ ਪ੍ਰਦਰਸ਼ਨ ਮਾਪਦੰਡਾਂ ਦੀ ਬੁਨਿਆਦੀ ਵਿਜ਼ੁਅਲਾਈਜ਼ੇਸ਼ਨ ਬਣਾਓ
3. **ਆਪਟੀਮਾਈਜ਼ੇਸ਼ਨ ਪ੍ਰਯੋਗ**: ਇੱਕ ਹੀ ਬੇਸ ਮਾਡਲ 'ਤੇ ਵੱਖ-ਵੱਖ ਕੁਆਂਟੀਜ਼ੇਸ਼ਨ ਪੱਧਰਾਂ ਦੇ ਪ੍ਰਭਾਵ ਨੂੰ ਮਾਪੋ

### ਮੱਧਮ ਪੱਧਰ ਦੇ ਪ੍ਰੋਜੈਕਟ (ਹਰ ਇੱਕ 3-4 ਘੰਟੇ)
4. **AI ਟੂਲਕਿਟ ਵਰਕਫਲੋ**: VS Code AI Toolkit ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਇੱਕ ਮਾਡਲ ਨੂੰ ਸ਼ੁਰੂ ਤੋਂ ਅੰਤ ਤੱਕ ਆਪਟੀਮਾਈਜ਼ ਅਤੇ ਡਿਪਲੌਇ ਕਰੋ
5. **Windows AI Foundry ਐਪਲੀਕੇਸ਼ਨ**: Phi Silica API ਅਤੇ NPU ਆਪਟੀਮਾਈਜ਼ੇਸ਼ਨ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਇੱਕ Windows ਐਪ ਬਣਾਓ
6. **ਕਰਾਸ-ਪਲੇਟਫਾਰਮ ਡਿਪਲੌਇਮੈਂਟ**: ਇੱਕ ਹੀ ਆਪਟੀਮਾਈਜ਼ ਮਾਡਲ ਨੂੰ Windows (OpenVINO) ਅਤੇ ਮੋਬਾਈਲ (.NET MAUI) 'ਤੇ ਡਿਪਲੌਇ ਕਰੋ
7. **ਫੰਕਸ਼ਨ ਕਾਲਿੰਗ ਏਜੰਟ**: Edge ਦ੍ਰਿਸ਼ਾਂ ਲਈ ਫੰਕਸ਼ਨ ਕਾਲਿੰਗ ਸਮਰਥਾ ਵਾਲਾ AI ਏਜੰਟ ਬਣਾਓ

### ਉੱਚ ਪੱਧਰ ਦੇ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਪ੍ਰੋਜੈਕਟ (ਹਰ ਇੱਕ 4-5 ਘੰਟੇ)
8. **OpenVINO ਆਪਟੀਮਾਈਜ਼ੇਸ਼ਨ ਪਾਈਪਲਾਈਨ**: NNCF ਅਤੇ GenAI ਟੂਲਕਿਟ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਪੂਰੀ ਮਾਡਲ ਆਪਟੀਮਾਈਜ਼ੇਸ਼ਨ ਲਾਗੂ ਕਰੋ
9. **SLMOps ਪਾਈਪਲਾਈਨ**: ਟ੍ਰੇਨਿੰਗ ਤੋਂ ਲੈ ਕੇ Edge ਡਿਪਲੌਇਮੈਂਟ ਤੱਕ ਪੂਰੀ ਮਾਡਲ ਲਾਈਫਸਾਈਕਲ ਲਾਗੂ ਕਰੋ
10. **ਮਲਟੀ-ਮਾਡਲ Edge ਸਿਸਟਮ**: Edge ਹਾਰਡਵੇਅਰ 'ਤੇ ਇੱਕਠੇ ਕੰਮ ਕਰਨ ਵਾਲੇ ਕਈ ਵਿਸ਼ੇਸ਼ ਮਾਡਲ ਡਿਪਲੌਇ ਕਰੋ
11. **MCP ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਸਿਸਟਮ**: ਟੂਲ ਇੰਟਰੈਕਸ਼ਨ ਲਈ Model Context Protocol ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਇੱਕ ਏਜੰਟਿਕ ਸਿਸਟਮ ਬਣਾਓ
